# תפיסת מצוף עגינה (Mooring)

- **קטגוריה:** תמרונים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** התקרבות למצוף עגינה.
- **צוות:** הגה + אחד בחרטום עם boat hook.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הכנה
- [ ] אשרו שהמצוף **מיועד למבקרים / מתאים לגודל הסירה שלכם**
- [ ] הכינו boat hook וחבל חרטום מנותב דרך ה-fairlead
- [ ] תדרכו את צוות החרטום וסכמו סימני יד
- [ ] שימו לב לרוח ולגאות — תכננו להתקרב **נגד החזק מבין השניים**

## התקרבות
1. [ ] התקרבו לאט נגד הרוח/הגאות כך שהסירה נעצרת באופן טבעי ליד המצוף
2. [ ] צוות החרטום מצביע על המצוף ברציפות כשהוא נעלם מתחת לחרטום
3. [ ] **עצרו את הסירה כשהמצוף בדיוק בחרטום** — אל תעברו אותו

## אבטחה
- [ ] תפסו את מצוף ההרמה / ה-strop; השחילו את החבל דרך הטבעת או קחו את ה-strop
- [ ] **קשרו ל-cleat חזק; אבטחו לפני שהסירה נסחפת לאחור**
- [ ] כווננו ל-bridle אם משתמשים בשני cleats בחרטום
- [ ] בדקו שהעומס נתפס והחבל אינו נשחק

## אשרו
- [ ] אשרו שאתם אוחזים; קחו transit
- [ ] הגדירו snubber/הגנת שחיקה ללינה
- [ ] **בדקו מרחב סיבוב יחסית לשכנים**

## הערות ומלכודות
- התקרבו נגד הכוח הדומיננטי (לרוב גאות בזרם חזק, רוח בגאות קלה).
- אם פספסתם, הקיפו ונסו שוב — אל תזנקו ואל תסכנו נפילת צוות למים.
