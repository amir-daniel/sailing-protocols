# Heaving-To (עצירה מיוצבת)

- **קטגוריה:** תמרונים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** לעצור/להאט ולייצב — מנוחה, reef, המתנה, או צליחת מזג אוויר.
- **צוות:** הגה + אחד (אפשרי לבד).

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## ביצוע (מ-close-hauled)
1. [ ] **בצעו tack מבלי לשחרר את ה-sheet של מפרש החרטום** — השאירו את ה-jib מהופך (backed)
2. [ ] כשהסירה עוברת, אל תיגעו ב-jib sheet
3. [ ] כשעל ה-tack החדש, **סובבו את ההגה עם הרוח (leeward) וקשרו אותו**
4. [ ] שחררו את ה-mainsheet עד שהסירה מתייצבת ונעצרת

## טרים לאיזון
- [ ] כווננו ראשי והגה כך שהסירה שוכבת כ-50–60° מהרוח, נסחפת לאט
- [ ] הפחיתו מפרש (reef) אם בעומס יתר בזמן heave-to
- [ ] אשרו היווצרות "slick" יציב ונוח מצד הרוח (windward)

## אשרו
- [ ] **בדקו את כיוון הסחיפה מול מרחב ים / מכשולים בצד המוגן (leeward)**
- [ ] נטרו מיקום; הסירה תסחף לאט (leeway)
- [ ] שמרו תצפית גם בזמן "חנייה"

## הערות ומלכודות
- כל סירה עושה heave-to אחרת — תרגלו בתנאים מתונים כדי להכיר את שלכם.
- מצוין לעצירת צהריים, reef, או הפסקה; גם טקטיקה תקפה למזג אוויר קשה.
