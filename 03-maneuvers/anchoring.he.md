# עגינה (Anchoring)

- **קטגוריה:** תמרונים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** עגינה לעצירת צהריים או ללינה.
- **צוות:** הגה + אחד בחרטום.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## בחרו את המקום
- [ ] בדקו במפה: עומק, **סוג אחיזה (חול/בוץ טוב; עשב/סלע גרוע)**, מכשולים
- [ ] העריכו מחסה מרוח ומ-swell (עכשיו ובתחזית)
- [ ] שימו לב למרחב סיבוב מול סירות אחרות והחוף
- [ ] חשבו scope: **לפחות 4:1 (שרשרת) / 6:1 (חבל) על העומק המרבי כולל גאות**

## הטילו את העוגן
1. [ ] הכינו את העוגן; אשרו שה-bitter end **קשור לסירה**
2. [ ] התקרבו לאט **נגד הרוח/הגאות**; עצרו מעל המקום שנבחר
3. [ ] הורידו (אל תזרקו) את העוגן כשהסירה מתחילה להיסחף לאחור
4. [ ] שחררו את ה-scope המחושב כשאתם נסוגים
5. [ ] תפסו את ה-rode; **הפליגו בעדינות לאחור כדי לנעוץ ולאשר שהוא אוחז**

## אשרו ונטרו
- [ ] קחו transits/bearings או הגדירו **התרעת סחיפת עוגן ב-GPS**
- [ ] הקימו snubber/bridle (שרשרת) כדי להפחית עומס פתאומי
- [ ] הציגו כדור עגינה (יום) / אור עגינה (לילה)
- [ ] **בדקו שוב את הנעיצה אחרי שהיא מתייצבת; בדקו שוב אם הרוח משתנה**

## הערות ומלכודות
- הפלגה עדינה לאחור לנעיצת העוגן היא השלב שאנשים מדלגים עליו — ולכן הם נסחפים.
- קחו בחשבון עליית גאות בחישוב ה-scope וגם עומק הסיבוב.
