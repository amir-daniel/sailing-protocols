# האטה, עצירה ו-Ferry Gliding

- **קטגוריה:** תמרונים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** שליטה במהירות/מיקום — התקרבות למעגנים, מצופים, תעלות, או החזקת מיקום.
- **צוות:** הגה + צוות לפי הצורך.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## האטה / עצירה במפרש
- [ ] **פנו אל הרוח כדי לשפוך כוח ולהאט**
- [ ] שחררו sheets כדי להוריד כוח מהמפרשים
- [ ] הפיכת מפרש החרטום (backing) מסייעת לעצור / לסובב את החרטום
- [ ] לעצירה מלאה, פנו head-to-wind ותנו למפרשים להתנופף (איבוד steerage — תכננו מראש)

## האטה / עצירה במנוע
- [ ] הפחיתו מצערת מוקדם; התנע נישא זמן רב במים
- [ ] **השתמשו בפרץ לאחור (astern) לעצירה; צפו ל-prop walk שיסובב את הירכתיים**
- [ ] דעו את כיוון ה-prop walk (לאיזה צד הירכתיים נבעטים ב-reverse)

## Ferry gliding (החזקת מיקום כנגד זרם)
- [ ] כוונו את החרטום **נגד הזרם (up-current)** בזווית
- [ ] כווננו סל"ד וזווית כך שהסירה נעה הצידה עם תנועה קדימה מועטה
- [ ] השתמשו בזה כדי להחזיק מיקום או לנוע לרוחב אל פונטון/קיר תעלה

## שליטה בטווח קרוב
- [ ] נועו **"as slow as possible, as fast as necessary"** ל-steerage
- [ ] שמרו steerage way — הגה זקוק לזרימת מים כדי לעבוד
- [ ] הכינו חבלים ו-fenders לפני שאתם מתחייבים

## הערות ומלכודות
- לסירת מפרש אין בלמים — תכננו את מרחק העצירה הרבה מראש.
- ב-reverse הסירה מגיבה באיטיות וה-prop walk שולט במהירות נמוכה; תרגלו זאת.
