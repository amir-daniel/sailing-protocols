# Tacking (פנייה נגד הרוח)

- **קטגוריה:** תמרונים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** סיבוב החרטום דרך הרוח כדי להחליף tack.
- **צוות:** הגה + אחד ב-sheets (אפשרי לבד).

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הכנה
- [ ] **בדקו תנועה ומכשולים ב-tack החדש**
- [ ] ודאו מהירות סירה מספקת כדי לעבור את הרוח
- [ ] הכינו את ה-sheet החדש (הרפוי); סיבוב אחד על ה-winch
- [ ] קִראו **"Ready about?"** — הצוות מאשר **"Ready!"**

## ביצוע
1. [ ] קִראו **"Lee-oh"** / "Tacking" וסובבו בחלקות אל הרוח
2. [ ] כשמפרש החרטום מתהפך (backs), שחררו את ה-sheet הישן בנקיות
3. [ ] משכו במהירות את ה-sheet החדש; winch לטרים
4. [ ] נווטו למסלול close-hauled החדש; אל תסובבו יתר על המידה

## אשרו
- [ ] טרמו מפרש חרטום וראשי ל-tack החדש
- [ ] **אשרו שה-sheet הישן פנוי ולא יסתבך**
- [ ] התייצבו על המסלול; בדקו שמהירות הסירה נבנית מחדש

## הערות ומלכודות
- איטי מדי בכניסה ל-tack = תקיעה "in irons"; שמרו מהירות לפני הסיבוב.
- אספו את הרפיון של ה-sheet הרפוי מוקדם כדי שעבודת ה-winch תהיה מהירה.
