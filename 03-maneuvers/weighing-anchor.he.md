# הרמת עוגן (Weighing Anchor)

- **קטגוריה:** תמרונים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** עזיבת מקום עגינה.
- **צוות:** הגה + אחד בחרטום/windlass.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הכנה
- [ ] תדרכו את הצוות; סכמו סימני יד (רעש מנוע/מרחק)
- [ ] התניעו את המנוע; **אשרו מי קירור**
- [ ] אחסנו את ה-snubber/bridle של העוגן מוכן לשחרור
- [ ] סדרו את הסיפון; סדרו את תא השרשרת כדי שהשרשרת תיאסף

## הרמה
1. [ ] הפליגו לאט קדימה לעבר העוגן בזמן שהצוות אוסף שרשרת
2. [ ] צוות החרטום מסמן להגה כיוון/כמות שרשרת
3. [ ] אספו עד שהשרשרת **"up and down"** (אנכית, מעל העוגן)
4. [ ] **שחררו את העוגן מהקרקעית** — הפליגו בעדינות או תנו למשקל הסירה לשחררו
5. [ ] אשרו שהעוגן מנותק מהקרקעית (הסירה מסתובבת חופשי)

## שליפה ואחסון
- [ ] הרימו את העוגן ל-bow roller; **שטפו בוץ/עשב בזמן העלייה**
- [ ] **אבטחו את העוגן ב-snubber/pin כדי שלא ישתחרר בזמן שיט**
- [ ] אשרו שה-windlass כבוי וה-clutch מאובטח
- [ ] הפליגו הרחק ממקום העגינה לפני הרמת מפרש

## עוגן מסובך (Fouled)
- [ ] אם לא משתחרר, תפסו את השרשרת והפליגו לאט מעליו כדי לשחררו
- [ ] אם כרוך על מכשול, השתמשו ב-**trip line / מצוף לעוגן** לשליפה
- [ ] אל תעמיסו יתר על ה-windlass — השתמשו במנוע/משקל הסירה, לא רק במנוע ה-windlass

## הערות ומלכודות
- הפליגו אל העוגן; אל תגררו את הסירה קדימה על ה-windlass — הוא אינו winch לגוף.
- אבטחו את העוגן כראוי לים; עוגן משתחרר-מעצמו שרץ החוצה בזמן שיט מסוכן.
