# Gybing (פנייה עם הרוח)

- **קטגוריה:** תמרונים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** סיבוב הירכתיים דרך הרוח (החלפה עם הרוח).
- **צוות:** הגה + אחד ב-mainsheet.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. ה-boom חוצה תחת עומס — שלטו בו.

## הכנה
- [ ] **בדקו תנועה ומכשולים ב-gybe החדש**
- [ ] **הזהירו את כל הצוות: "heads down" — הרחק מ-mainsheet ומנתיב ה-boom**
- [ ] אספו את ה-mainsheet כדי למרכז את ה-boom (gybe מבוקר)
- [ ] הכינו את ה-sheet החדש של מפרש החרטום / הכינו pole אם עף spinnaker
- [ ] קִראו **"Ready to gybe?"** — הצוות מאשר **"Ready!"**

## ביצוע
1. [ ] קִראו **"Gybe-oh"** ופנו עם הרוח בחלקות דרך dead downwind
2. [ ] כשהירכתיים חוצים את הרוח, **שלטו במעבר ה-boom — אל תתנו לו לחבוט**
3. [ ] שחררו את ה-mainsheet מיד בצד החדש
4. [ ] העבירו את מפרש החרטום; טרמו בצד החדש

## אשרו
- [ ] נווטו למסלול החדש; שחררו vang/sheet לטרים נכון
- [ ] בדקו שה-preventer מוקם מחדש אם היה בשימוש
- [ ] **אשרו שאין צוות או חבלים בנתיב תנועת ה-boom**

## הערות ומלכודות
- "crash gybe" לא מבוקר עלול לפצוע צוות ולהזיק לכלונסאות — תמיד אספו
  sheet קודם ושלטו במעבר ה-boom.
- הקימו **boom preventer** עם הרוח כדי למנוע gybes בשוגג.
