# חירום רפואי

- **קטגוריה:** מצבי חירום
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** פציעה או מחלה חמורה על הסירה.
- **צוות:** כל הידיים; חלקו תפקידים.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. אינו תחליף להכשרת עזרה ראשונה.

## קחו פיקוד והעריכו (סקירה ראשונית)
1. [ ] **ודאו שהזירה בטוחה** למחלצים (התחברו בחבל, אבטחו את הסירה)
2. [ ] העריכו את הנפגע — **DR ABC**:
   - **D**anger (סכנה), **R**esponse (תגובה)
   - **A**irway — נתיב אוויר פתוח ופנוי
   - **B**reathing — הביטו, הקשיבו, הרגישו
   - **C**irculation — דופק, דימום משמעותי
3. [ ] **עצרו דימום קטסטרופלי מיד** (לחץ ישיר)
4. [ ] התחילו **CPR** אם אינו נושם כרגיל

## ייצוב
- [ ] טפלו קודם באיומי חיים: דימום, נתיב אוויר, נשימה
- [ ] קבעו/שתקו פציעות עמוד שדרה/שברים חשודות
- [ ] שמרו על הנפגע חם; טפלו בהלם; הרגיעו
- [ ] תעדו תצפיות, זמנים וכל תרופה שניתנה

## קבלו עזרה
- [ ] **בקשו ייעוץ רפואי: DSC + PAN-PAN, או Mayday אם מסכן חיים**
- [ ] מסרו את מצב הנפגע, סימנים חיוניים ואת מיקומכם
- [ ] בקשו פינוי / הסטה אם הומלץ
- [ ] היערכו להעברה למסוק/סירת הצלה אם תואם

## לאחר מכן
- [ ] המשיכו לנטר; שמרו רישומים עבור הצוות הרפואי
- [ ] תמכו בשאר הצוות
- [ ] תעדו את האירוע

## הערות ומלכודות
- החזיקו ערכת עזרה ראשונה מקיפה ומדריך רפואי לים; הכירו את מצבי הרקע
  והתרופות של הצוות לפני היציאה.
- ב-VHF ניתן לבקש ייעוץ רפואי מבלי שזו תהיה קריאת מצוקה מלאה.
