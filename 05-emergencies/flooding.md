# Flooding / Hull Breach

- **Category:** Emergencies
- **Style:** READ–DO
- **When to use:** Water ingress rising in the bilge.
- **Crew:** All hands.

> Killer items are shown in **bold**.

## Find the source
1. [ ] **Start all bilge pumps** (electric + manual) and detail someone to pump
2. [ ] Taste/inspect: salt = ingress; check likely sources fast:
   - Skin fittings / seacocks / hoses
   - Stern gland / stuffing box
   - Log/depth transducer
   - Rudder stock, keel bolts
   - Damaged hull / impact point

## Stop / slow the ingress
- [ ] **Close the relevant seacock** if source is a through-hull/hose
- [ ] Soft-wood bung into a failed skin fitting or hole
- [ ] Stuff/cushion + brace over a breach; use a collision mat / sail outside
- [ ] Slow the boat or change tack/heel to lift the hole above the waterline

## If not controllable
- [ ] **Send DSC distress + Mayday** with position and nature (taking on water)
- [ ] Rig emergency/engine-driven pump if available
- [ ] Prepare liferaft and grab bag (see abandon-ship)
- [ ] Head for shallow water/beaching if sinking is imminent

## After
- [ ] Keep monitoring the bilge and pumps
- [ ] Effect a more permanent repair; check the pump's ability to keep up
- [ ] Log the event

## Notes & gotchas
- Keep soft-wood bungs tied at every seacock — the fastest fix for a failed fitting.
- A pump that just keeps pace buys time; the priority is stopping the ingress.
