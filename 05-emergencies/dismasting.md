# Dismasting

- **Category:** Emergencies
- **Style:** READ–DO
- **When to use:** Mast has come down or is broken over the side.
- **Crew:** All hands.

> Killer items are shown in **bold**.

## Immediate safety
1. [ ] **Account for all crew — check no one is injured or trapped in rigging**
2. [ ] Start the engine **only after confirming no lines are in the water** (prop)
3. [ ] Note the wreck is now alongside — protect the hull from being holed

## Secure the wreckage
- [ ] Assess: can the rig be salvaged, or must it be cut away?
- [ ] If it is holing the hull / dangerous: **cut it away** using bolt croppers / rigging knife
- [ ] Cut/release standing rigging methodically; retain the boom/spar if safe to keep
- [ ] Recover valuable gear (sails, spar sections) if conditions allow

## Get underway / call for help
- [ ] Rig a jury mast / jury rig if you can, to make way or steady the boat
- [ ] Motor if safe and fuel allows
- [ ] **Send a DSC alert / PAN-PAN (or Mayday if in danger)** with position

## After
- [ ] Treat any injuries
- [ ] Secure loose gear; make the boat seaworthy
- [ ] Log the event and plan for landfall / tow

## Notes & gotchas
- Keep bolt croppers and a serrated rigging knife accessible — you may need to
  cut away fast to save the hull.
- Lines and wire in the water will foul the prop; check before engaging gear.
