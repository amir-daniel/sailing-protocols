# כשל במנוע

- **קטגוריה:** מצבי חירום
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** המנוע נעצר או אינו מותנע כשצריך.
- **צוות:** סקיפר + אחד.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## שיקול שיט מיידי
1. [ ] **העריכו מרחב ים ומכשולים; היו מוכנים להפליג או לעגון**
2. [ ] אם קרוב לסכנה, הרימו מפרש או הפילו עוגן כדי להחזיק מיקום

## אבחון (הגורמים הנפוצים)
- [ ] **דלק:** מספיק במיכל? ברז הדלק פתוח? אוויר במערכת (bleed)? מסנן חסום?
- [ ] **אוויר/קירור:** raw-water intake פתוח? עשב/פלסטיק על ה-intake? impeller תקין? התרעת התחממות?
- [ ] **חשמל:** מצבר/isolator דולק? Kill-cord/בקרת עצירה אופסה? חיבור רופף?
- [ ] **prop מסובך:** חבל או עשב סביב המדחף (בדקו לפני התנעה מחדש)

## פעולה
- [ ] נקו/החליפו את הרכיב התקול (bleed לדלק, החלפת מסנן, החלפת impeller)
- [ ] **אל תכניסו הילוך מחדש אם ייתכן שחבל סביב ה-prop** — בדקו קודם
- [ ] בהתחממות יתר: עצרו, תנו להתקרר, אתרו את תקלת הקירור לפני התנעה מחדש

## אם לא נפתר
- [ ] המשיכו במפרש לעבר יעד בטוח
- [ ] קראו לגרירה (PAN-PAN) אם אינכם יכולים להגיע לנמל בבטחה

## לאחר מכן
- [ ] תעדו את התסמינים ואת הפתרון
- [ ] חדשו מלאי מתכלים שנוצלו (מסננים, impeller)

## הערות ומלכודות
- רוב כשלי הדיזל קשורים בדלק: מסנן מלוכלך, אוויר, או מיכל ריק/מזוהם.
- החזיקו ודעו כיצד להרכיב impeller ומסנן דלק חלופיים.
