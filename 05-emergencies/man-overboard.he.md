# אדם בים (MOB)

- **קטגוריה:** מצבי חירום
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** ברגע שאדם נופל למים.
- **צוות:** כל הידיים.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. פעלו מיד — כל שנייה קובעת.

## פעולות מיידיות (10 השניות הראשונות)
1. [ ] **צעקו "MAN OVERBOARD!"** כדי להתריע לכל הצוות
2. [ ] **השליכו ציפה** (danbuoy, מצוף פרסה, כרית) לכיוון הנופל
3. [ ] **לחצו על כפתור ה-MOB** ב-GPS/מכשיר הניווט כדי לסמן את המיקום
4. [ ] **הקצו צופה ייעודי** שיצביע על הנופל ולא יסיר ממנו את המבט לרגע

## תמרון חזרה
- [ ] שימו לב לכיוון הרוח; התחילו חזרה (tack/gybe, או במנוע)
- [ ] הפעילו את המנוע אם בטוח; **בדקו שאין חבלים במים לפני הכנסה להילוך**
- [ ] התקרבו מכיוון מורד-הרוח ועצרו את הסירה לצד הנופל

## התרעה (אם החילוץ אינו מיידי)
- [ ] שדרו **DSC distress** ושדרו **Mayday** ב-VHF Ch 16
- [ ] מסרו מיקום, "man overboard", מספר האנשים במים ותיאור
- [ ] הקצו אנשי צוות לסיוע, ובמידת הצורך הוסיפו אמצעים

## חילוץ
- [ ] צרו ושמרו על קשר; העבירו חבל/lifesling אל הנופל
- [ ] **אין להכניס מחלץ למים אלא כמוצא אחרון, וכשהוא קשור בחבל ביטחון**
- [ ] חלצו בעזרת lifesling/מעלן, סולם עלייה או רצועת הרמה
- [ ] העלו לסיפון **במאוזן אם ניתן** (מפחית התמוטטות לאחר טבילה)

## לאחר החילוץ
- [ ] טפלו בהיפותרמיה, הלם ופציעות; נטרו נשימה
- [ ] שמרו על הנפגע חם ורגוע; בקשו ייעוץ רפואי דרך ה-VHF
- [ ] עדכנו/בטלו את ה-Mayday לפי העניין
- [ ] תעדו את האירוע ותחקרו את הצוות

## הערות ומלכודות
- תפקידו היחיד של הצופה הוא להצביע — הקצו אותו מיד ואל תשנו את התפקיד.
- חילוץ נפגע כבד וקר מהמים הוא החלק הקשה ביותר: הכינו יתרון מכני
  (מעלן + winch, lifesling) מוקדם.
