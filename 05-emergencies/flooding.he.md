# הצפה / פריצת גוף הסירה

- **קטגוריה:** מצבי חירום
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** חדירת מים עולה ב-bilge (תחתית הסירה).
- **צוות:** כל הידיים.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## אתרו את המקור
1. [ ] **הפעילו את כל משאבות ה-bilge** (חשמלית + ידנית) והקצו מישהו לשאוב
2. [ ] טעמו/בדקו: מלוח = חדירת מי ים; בדקו במהירות את המקורות הסבירים:
   - Skin fittings / seacocks / צינורות
   - Stern gland / stuffing box
   - Log/depth transducer
   - ציר ההגה (rudder stock), ברגי הכובד (keel bolts)
   - גוף פגוע / נקודת פגיעה

## עצרו / האטו את החדירה
- [ ] **סגרו את ה-seacock הרלוונטי** אם המקור הוא through-hull/צינור
- [ ] הכניסו bung מעץ רך לתוך skin fitting או חור שכשל
- [ ] דחסו/רפדו + חזקו מעל הפרצה; השתמשו ב-collision mat / מפרש מבחוץ
- [ ] האטו את הסירה או שנו tack/הטיה כדי להרים את החור מעל קו המים

## אם לא ניתן לשלוט
- [ ] **שדרו DSC distress + Mayday** עם מיקום ומהות (חודרים מים)
- [ ] הפעילו משאבת חירום/משאבה מונעת-מנוע אם קיימת
- [ ] הכינו liferaft ו-grab bag (ראו abandon-ship)
- [ ] נווטו למים רדודים/עלייה על החוף אם ההטבעה מיידית

## לאחר מכן
- [ ] המשיכו לנטר את ה-bilge ואת המשאבות
- [ ] בצעו תיקון קבוע יותר; בדקו שהמשאבה עומדת בקצב
- [ ] תעדו את האירוע

## הערות ומלכודות
- החזיקו bungs מעץ רך קשורים ליד כל seacock — התיקון המהיר ביותר ל-fitting שכשל.
- משאבה שרק עומדת בקצב קונה זמן; העדיפות היא לעצור את החדירה.
