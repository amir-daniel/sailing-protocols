# נטישת כלי השיט

- **קטגוריה:** מצבי חירום
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** מוצא אחרון — רק כאשר הסירה בוודאות אבודה.
- **צוות:** כל הידיים.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. **עלו מעלה אל ה-liferaft** — לעולם אל תעזבו
> את הסירה עד שאתם חייבים.

## החלטה
- [ ] **נטשו רק כאשר היאכטה כבר לא יכולה לתמוך בכם** (טובעת/בוערת/מתפרקת)
- [ ] היאכטה כמעט תמיד liferaft טוב יותר מהרפסודה — היו בטוחים

## התרעה
1. [ ] **שדרו Mayday: DSC distress + קול**, מיקום, POB (מספר נפשות), מהות
2. [ ] הפעילו **EPIRB**; קחו את ה-**PLB** עליכם

## הכנה (אם הזמן מאפשר)
- [ ] כולם באפודי הצלה **ובהגנה תרמית / drysuits**
- [ ] Grab bag: מים, רקטות, VHF ידני, PLB, עזרה ראשונה, פנס, ביגוד חם
- [ ] קחו מים נוספים וכל רקטה נוספת
- [ ] שגרו את ה-liferaft **קשור לסירה ב-painter שלו**, בצד המוגן מהרוח (leeward)

## עלייה לרפסודה
- [ ] **עלו ישירות / עלו מעלה — הישארו יבשים ככל האפשר**
- [ ] הכניסו את כולם; ספרו את כל הנפשות (POB)
- [ ] כשכולם בפנים, **חתכו את ה-painter** כדי להתרחק מכלי טובע
- [ ] פרסו את ה-drogue; סגרו את החופה; קחו כדורים נגד מחלת ים

## הישרדות
- [ ] סדר עדיפויות: **הגנה, מיקום (Location), מים, מזון** (בסדר הזה)
- [ ] שמרו תצפית; חסכו ברקטות עד שההצלה קרובה
- [ ] קצבו מים; שמרו את הרפסודה יבשה ואת כולם חמים; שמרו על מורל

## הערות ומלכודות
- חברו (clip) את ה-grab bag ואת ה-painter כדי שדבר לא יאבד בבהלה.
- אל תנפחו אפודי הצלה *בתוך* הרפסודה; תתפסו מקום ותסכנו הסתבכות.
