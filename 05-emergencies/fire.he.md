# שריפה

- **קטגוריה:** מצבי חירום
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** בכל שריפה או עשן על הסירה.
- **צוות:** כל הידיים.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. פעלו מהר; אש מכפילה את עצמה במהירות.

## מיידי
1. [ ] **צעקו "FIRE" ואתרו את המקור**
2. [ ] **נתקו את מקור הדלק/האנרגיה** לאש:
   - מטבח/גז: סגרו את הגז בבלון וב-solenoid
   - מנוע: עצרו את המנוע, נתקו את הדלק
   - חשמל: נתקו את ה-isolator של המצבר / המעגל התקול
3. [ ] העלו את כל הצוות הלא-חיוני לסיפון עם אפודי הצלה, בצד הפונה לרוח (upwind)

## כיבוי
- [ ] כוונו את המטף אל **בסיס** הלהבות, בתנועת טאטוא
- [ ] שריפת מטבח/בישול: חנקו עם שמיכת אש
- [ ] **שריפת מנוע: השתמשו במערכת הקבועה או בפתח הייעודי — אל תפתחו את תא המנוע לרווחה** (מזין חמצן)
- [ ] סגרו פתחים/צוהרים כדי למנוע אספקת אוויר לאש, ככל הניתן

## אם השריפה לא נשלטת
- [ ] **שדרו DSC distress + Mayday** עם מיקום ומהות (שריפה)
- [ ] הכינו liferaft ו-grab bag; היו מוכנים לנטוש (ראו abandon-ship)
- [ ] כבו ממיקום שמשאיר את נתיב המילוט שלכם פתוח

## לאחר מכן
- [ ] אווררו; שימו לב להתלקחות חוזרת
- [ ] בדקו נזק לגוף הסירה/מבני ונפגעים
- [ ] טפלו בכוויות; בקשו ייעוץ רפואי
- [ ] תעדו את האירוע

## הערות ומלכודות
- שלילת החמצן והדלק חשובה יותר מהמטף עצמו.
- לעולם אל תפתחו תא מנוע לחלוטין כדי לכבות שריפה — השתמשו בפתח הכיבוי הייעודי.
