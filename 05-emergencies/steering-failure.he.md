# כשל בהגה

- **קטגוריה:** מצבי חירום
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** אובדן היגוי (הגה/כבל, quadrant, או ההגה עצמו).
- **צוות:** כל הידיים.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הערכה
1. [ ] שימו לב למיקום, לתנועה ולמכשולים; **הורידו כוח מהמפרשים כדי להאט את הסירה**
2. [ ] קבעו את סוג הכשל:
   - כבל/quadrant/מנגנון קישור (מכני)
   - Rudder stock / מיסב / אובדן ההגה (מבני)

## החזירו שליטה
- [ ] **הרכיבו את ה-emergency tiller** על ה-rudder stock (דעו היכן הוא מאוחסן)
- [ ] אם הקישור כשל אך ההגה תקין, נווטו ישירות דרך ה-emergency tiller
- [ ] אם ההגה אבד/תקוע:
  - אזנו מפרשים כדי לנווט (טרים בין המפרש הראשי למפרש החרטום)
  - הקימו **drogue/warp bridle מרבעי הירכתיים** לניווט
  - השתמשו במשוט/קרש רצפה קשור כלוח היגוי במידת הצורך

## נהלו את המצב
- [ ] האטו; נווטו למרחב פתוח או לנמל מקלט
- [ ] **PAN-PAN (או Mayday אם בסכנה)** עם מיקום אם אינכם יכולים לנווט בבטחה
- [ ] שקלו עגינה אם קרוב למכשול ובעומק מספיק רדוד

## לאחר מכן
- [ ] בדקו ונסו לתקן כשבטוח
- [ ] תעדו את האירוע

## הערות ומלכודות
- תרגלו הרכבת ה-emergency tiller *לפני* שתצטרכו אותו — הגישה אליו
  לרוב מסורבלת תחת עומס.
- איזון מפרשים לבדו יכול להחזיק כיוון בסירות רבות; שחררו את המפרש הראשי כדי לפנות עם הרוח.
