# שבירת תורן (Dismasting)

- **קטגוריה:** מצבי חירום
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** התורן קרס או נשבר אל מעבר לדופן.
- **צוות:** כל הידיים.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## בטיחות מיידית
1. [ ] **ספרו את כל הצוות — ודאו שאיש אינו פצוע או לכוד בכלונסאות (rigging)**
2. [ ] הפעילו את המנוע **רק לאחר שווידאתם שאין חבלים במים** (prop)
3. [ ] שימו לב שהגרוטאה עתה צמודה לדופן — הגנו על הגוף מפני ניקוב

## אבטחו את ההריסות
- [ ] העריכו: האם ניתן להציל את התורן, או שיש לחתוך אותו?
- [ ] אם הוא מנקב את הגוף / מסוכן: **חתכו אותו** בעזרת bolt croppers / סכין rigging
- [ ] חתכו/שחררו את ה-standing rigging בשיטתיות; שמרו את ה-boom/spar אם בטוח
- [ ] הצילו ציוד יקר ערך (מפרשים, חלקי spar) אם התנאים מאפשרים

## צאו לדרך / קראו לעזרה
- [ ] הקימו jury mast / jury rig אם תוכלו, כדי להתקדם או לייצב את הסירה
- [ ] הפעילו מנוע אם בטוח והדלק מאפשר
- [ ] **שדרו DSC alert / PAN-PAN (או Mayday אם בסכנה)** עם מיקום

## לאחר מכן
- [ ] טפלו בפציעות
- [ ] אבטחו ציוד רופף; הפכו את הסירה לכשירת שיט
- [ ] תעדו את האירוע ותכננו הגעה לחוף / גרירה

## הערות ומלכודות
- החזיקו bolt croppers וסכין rigging משוננת בהישג יד — ייתכן שתצטרכו
  לחתוך במהירות כדי להציל את הגוף.
- חבלים וכבלים במים יסתבכו ב-prop; בדקו לפני הכנסה להילוך.
