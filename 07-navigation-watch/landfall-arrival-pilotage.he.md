# הגעה לחוף וניווט כניסה (Pilotage)

- **קטגוריה:** ניווט ומשמרת
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** התקרבות לנמל/מקום עגינה לאחר מעבר.
- **צוות:** סקיפר/נווט + צופה.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. תכננו את הכניסה *לפני* שאתם מגיעים, רצוי באור יום.

## תכננו לפני ההגעה
- [ ] למדו את הנמל: **כניסה, עומקים, חלון גישה לפי גאות, מכשולים, סימני הובלה**
- [ ] ציינו את **גובה/זרם הגאות** לזמן ההגעה (מספיק עומק? גאות נוחה?)
- [ ] זהו buoyage/קווי הובלה ו-**transit** ברור להיכנס עליו
- [ ] ציינו ערוץ VHF של הנמל, שעות, וכל אות תנועה
- [ ] החזיקו **תוכנית pilotage / סקיצה**: מסלולים, סימנים, clearing bearings, אזורי סכנה
- [ ] תכננו **ביטול/פנייה בטוחים** אם זה לא נראה טוב

## בהתקרבות
- [ ] קבעו מיקום; אשרו שאתם מזהים את הסימנים ה**נכונים** (ודאו, אל תניחו)
- [ ] **התקשרו למרינה/לנמל** לקבלת מעגן והוראות
- [ ] הפחיתו מפרש / הכינו את המנוע בעוד מועד; **אשרו מי קירור**
- [ ] הציבו צופה; שימו לב לעומק מול חישוב הגאות שלכם
- [ ] השתמשו ב-clearing bearings/transits כדי להישאר במים בטוחים

## היכנסו ועגנו
- [ ] הרכיבו fenders וחבלים **בשני הצדדים** לפני הכניסה (ראו coming-alongside)
- [ ] עקבו אחר ה-buoyage; תנו זכות קדימה לפי COLREGs וכללים מקומיים; שמרו ימינה
- [ ] התקדמו במהירות איטית ומבוקרת; היו מוכנים לבטל ולהקיף

## הערות ומלכודות
- **ודאו** את הסימנים מול המפה — טעות בזיהוי מצוף גורמת לעליות שרטון רבות.
- אם לא בטוחים, האטו, המתינו, או חכו לגאות/אור יום. אין פרס על כניסה מסוכנת.
