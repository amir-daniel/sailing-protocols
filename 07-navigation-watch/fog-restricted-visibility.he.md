# ערפל / ראות מוגבלת

- **קטגוריה:** ניווט ומשמרת
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** הראות יורדת (ערפל, גשם כבד, שלג).
- **צוות:** כל הידיים ערניים.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. פעלו לפי COLREGs לראות מוגבלת.

## מיידי
1. [ ] **קבעו ותעדו את מיקומכם עכשיו** בזמן שאתם עדיין יכולים
2. [ ] **הדליקו אורות ניווט**
3. [ ] הציבו צופים נוספים — **הקשיבו כמו גם הביטו**
4. [ ] הפחיתו ל**מהירות בטוחה** לראות

## אמצעי בטיחות
- [ ] השמיעו את אות הערפל הנכון במרווחים (מתחת ל-2 דק'):
  - מפרש: ארוכה אחת + שתיים קצרות
  - במנוע: ארוכה אחת
- [ ] **הכניסו את כל הצוות לאפודי הצלה**
- [ ] שקלו הרמת radar reflector (אם לא קבוע)
- [ ] השתמשו ב-radar/AIS לזיהוי תנועה; נטרו CPA/TCPA
- [ ] נטרו VHF Ch 16

## החלטות ניווט
- [ ] **שקלו לעזוב/להימנע מנתיבי שיט; פנו למים רדודים הרחק מתנועה**
- [ ] נווטו בזהירות עם GPS + DR; שימו לב לעומק כבדיקה צולבת
- [ ] אם קרוב למכשולים ולא בטוחים, שקלו עגינה במים בטוחים ורדודים

## הערות ומלכודות
- אל תסתמכו רק על radar/AIS — כלי קטן עשוי לא להופיע; שמרו צופים אנושיים.
- האטו: היכולת לעצור בתוך טווח הראות היא כל העניין.
