# הגעה / התייצבות צמוד לרציף

- **קטגוריה:** עגינה ורציף
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** התייצבות צמוד לפונטון, קיר, או סירה אחרת.
- **צוות:** הגה + צוות רציף.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הכנה
- [ ] התקשרו מראש / אשרו את המעגן ולאיזה צד
- [ ] **הרכיבו fenders בצד העגינה בגובה הנכון; חבלים מוכנים**
- [ ] הכינו חבל חרטום, חבל ירכתיים ו-springs, מנותבים מחוץ להכול
- [ ] תדרכו את הצוות: מי יורד (רק כשבטוח), איזה חבל ראשון
- [ ] **העריכו רוח/גאות** — תכננו להתקרב נגד החזק

## התקרבות
1. [ ] התקרבו לאט בזווית רדודה נגד הרוח/הגאות
2. [ ] שלטו במהירות — **"as slow as possible, as fast as necessary"**
3. [ ] הביאו את הסירה במקביל ועצרו צמוד
4. [ ] **רדו לחוף ברוגע — לעולם אל תקפצו; אל תבלמו עם ידיים/רגליים**

## אבטחה
- [ ] העבירו spring או את החבל הראשון כדי לעצור תנועה
- [ ] חברו חרטום, ירכתיים ו-springs; כווננו לטווח הגאות
- [ ] הציבו fenders נכון; הוסיפו הגנת spring/שחיקה
- [ ] **השאירו רפיון לעליית/ירידת גאות אם מתייבשים או טווח גדול**

## אשרו
- [ ] מנוע כבוי; shore power / בדיקות לפי הצורך
- [ ] חבלים מסודרים, הגנת שחיקה
- [ ] Fenders מכווננים לנקודות המגע בפועל

## הערות ומלכודות
- התקרבו נגד החזק מבין רוח/גאות לשליטה מרבית במהירות נמוכה.
- Spring אמצע-הסירה ראשון הוא החבל היחיד הקל ביותר לשליטה בסירה.
