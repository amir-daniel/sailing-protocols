# יציאה מהמעגן

- **קטגוריה:** עגינה ורציף
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** עזיבת מעגן במרינה או צמוד לרציף.
- **צוות:** הגה + צוות רציף.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## לפני שחרור החבלים
- [ ] השלימו את בדיקות טרום-ההתנעה/המנוע (ראו רשימת המנוע)
- [ ] **העריכו כיוון ועוצמת רוח וגאות** במעגן
- [ ] תדרכו את הצוות: סדר החבלים, מי יורד/עולה, fending
- [ ] Fenders ממוקמים ליציאה המתוכננת ולכל pivot
- [ ] סכמו את התוכנית (יציאה בחרטום/ירכתיים, spring off, ferry glide)

## שחררו את החבלים
- [ ] המנוע פועל, **מי קירור מאושרים**, הילוך נבדק קדימה/אחורה
- [ ] הסירו חבלים מיותרים; השאירו slip line / spring לשליטה
- [ ] **שמרו ידיים ואצבעות הרחק מ-cleats ומהרווחים בין הגוף לפונטון**
- [ ] השתמשו ב-spring כדי לסובב את החרטום/הירכתיים החוצה אם רוח/גאות מצמידות אתכם

## יציאה
- [ ] שחררו את החבל האחרון רק כשנתיב היציאה פנוי
- [ ] הפליגו החוצה בחלקות; שימו לב שהירכתיים נעים לכיוון הפונטון
- [ ] **שלפו את כל החבלים וה-fenders לאחר שהתרחקתם** (לפני שיסתבכו ב-prop)

## אשרו
- [ ] חבלים כרוכים, fenders מאוחסנים
- [ ] רחוק מהמרינה; התייצבו על המסלול

## הערות ומלכודות
- Slip line (מושחל בחזרה לסירה) מאפשר שחרור מהסירה — אף אחד לא נשאר על הרציף.
- רוח/גאות מצמידות אתכם? השתמשו ב-spring כמנוף לשחרור במקום להפעיל כוח נגדן.
