# עגינה ים-תיכונית (Med Mooring) (stern-to / bows-to)

- **קטגוריה:** עגינה ורציף
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** עגינה בסגנון ים-תיכוני לרציף עם עוגן או lazy line.
- **צוות:** הגה + חרטום + צוות ירכתיים (מינימום 3 מועיל).

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הכנה
- [ ] החליטו **stern-to** (עלייה קלה יותר) או **bows-to** (יותר פרטיות/הגנת prop)
- [ ] הרכיבו fenders **בשני הצדדים**; הכינו שני חבלי ירכתיים (או חרטום) לרציף
- [ ] הכינו את העוגן עם scope רב, **או** אתרו את ה-lazy line
- [ ] תדרכו את הצוות וסכמו סימנים; שימו לב לרוח/רוח צד

## התקרבות והטלה (שיטת עוגן, stern-to)
1. [ ] היישרו בניצב לרציף, בהתחשב ברוח הצד
2. [ ] **הטילו את העוגן הרחק (כ-4–5 אורכי סירה), משחררים שרשרת בזמן הנסיגה**
3. [ ] הפליגו לאחור לעבר הרציף, מנווטים בפרצי כוח
4. [ ] שמרו מתח מספיק בשרשרת העוגן כדי להישאר ישרים

## אבטחה
- [ ] העבירו **חבלי ירכתיים לרציף** וקשרו
- [ ] מתחו את העוגן כדי להחזיק את החרטום מרוחק; כווננו חבלי ירכתיים
- [ ] **שמרו את הסירה מרוחקת מהרציף — לעולם אל תתנו לירכתיים/הגה לפגוע**
- [ ] הרכיבו passerelle/קרש לעלייה

## וריאנט lazy-line
- [ ] תפסו את ה-lazy line ברציף, **הובילו אותו קדימה וקשרו בחרטום**
- [ ] מתחו אותו כדי להחזיק את החרטום מרוחק במקום עוגן

## אשרו
- [ ] כווננו את כל החבלים לעומס אחיד ומרווח
- [ ] בדקו שעוגני השכנים אינם מוצלבים עם שלכם
- [ ] Fenders ממוקמים למגע בפועל עם השכנים

## הערות ומלכודות
- עוגנים מוצלבים הם הבלגן הקלאסי של Med-mooring — הטילו ישר וציינו לאן מובילות שרשראות השכנים.
- הגנו על ההגה/prop: עצרו לפני שהירכתיים נוגעים; השתמשו בחבל ירכתיים ארוך במקום לנסוג בכוח על הרציף.
