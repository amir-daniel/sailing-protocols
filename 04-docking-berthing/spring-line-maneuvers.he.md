# תמרוני Spring-Line

- **קטגוריה:** עגינה ורציף
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** יציאה/עגינה במקומות צרים או כשרוח/גאות מצמידות אתכם.
- **צוות:** הגה + אחד ב-spring.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הרעיון
- **Spring** הוא חבל מנותב קדימה-אחורה, שכשעובד נגד דחף המנוע,
  ממנף את החרטום או הירכתיים פנימה או החוצה. הוא מאפשר לסובב את הסירה באורכה שלה.

## הוצאת הירכתיים (כדי לצאת קדימה)
1. [ ] הקימו **bow spring** (מהחרטום, אחורה ל-cleat ברציף) כ-slip line
2. [ ] הציבו fenders בכתף החרטום (נקודת ה-pivot/מגע)
3. [ ] **אספו את כל שאר החבלים**
4. [ ] הפליגו **בעדינות קדימה נגד ה-spring** — הירכתיים נעים החוצה
5. [ ] כשהזווית מספקת, שחררו את ה-spring וצאו לאחור (reverse)

## הוצאת החרטום (כדי לצאת לאחור)
1. [ ] הקימו **stern spring** (מהירכתיים, קדימה ל-cleat ברציף) כ-slip line
2. [ ] Fenders בכתף הירכתיים
3. [ ] **אספו את כל שאר החבלים**
4. [ ] הפליגו **בעדינות לאחור נגד ה-spring** — החרטום נע החוצה
5. [ ] שחררו את ה-spring וצאו קדימה

## אשרו
- [ ] **אשרו שה-spring הוא slip line שניתן לשחרר מהסירה**
- [ ] שלפו spring ו-fenders לאחר שהתרחקתם
- [ ] שימו לב שהקצה המסתובב לא נוגע ברציף ללא fender

## הערות ומלכודות
- שמרו סל"ד נמוך — ה-spring עושה את העבודה, לא כוח גס.
- Fending טוב בנקודת ה-pivot הוא מה שמגן על הגוף; מקמו אותו במדויק.
