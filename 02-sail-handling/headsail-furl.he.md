# הנפה / גלילה של מפרש החרטום (roller furling genoa)

- **קטגוריה:** טיפול במפרשים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** פרישה או אחסון של מפרש חרטום עם roller-furling.
- **צוות:** אחד בקוקפיט (אפשרי לבד).

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## פרישה
- [ ] פנו ל-reach כדי להפחית עומס והצלפה
- [ ] שחררו את ה-furling line, תוך שמירה על **מתח קל כדי שלא ישתחרר יתר על המידה**
- [ ] משכו את ה-sheet המוגן (leeward) כדי לפרוש את המפרש
- [ ] טרמו למסלול; כווננו את ה-sheet lead / מיקום ה-car
- [ ] קבעו את ה-furling line תוך השארת מספר סיבובים

## קיצור (גלילה חלקית)
- [ ] שחררו מעט את ה-sheet
- [ ] אספו furling line לגודל הרצוי
- [ ] **הזיזו את ה-sheet car קדימה כדי לשמור על איזון foot/leech**
- [ ] טרמו מחדש

## גלילה / אחסון
- [ ] פנו ל-reach; שחררו את ה-sheet כדי שהמפרש יאבד כוח
- [ ] משכו את ה-furling line בהדרגה, תוך שמירה על **מתח ב-sheet הרפוי** לגלילה הדוקה
- [ ] גללו לחלוטין; **סיימו כשה-UV strip בחוץ**
- [ ] קבעו את ה-furling line בחוזקה; סדרו את ה-sheets

## הערות ומלכודות
- גלילה רופפת ברוח חזקה עלולה להיפרש ולהעצים יתר על המידה את הסירה — גללו הדוק וקבעו חזק.
- לעולם אל תכריחו furler תקוע; אתרו את ה-riding turn או ה-override קודם.
