# הורדה וקיפול המפרש הראשי

- **קטגוריה:** טיפול במפרשים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** התקרבות לנמל / סיום שיט / מעבר למנוע.
- **צוות:** הגה + אחד ליד המפרש.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הכנה
- [ ] אזור פנוי להנעה במנוע; **המנוע מותנע ובהילוך לפני ההורדה (steerage)**
- [ ] כווננו topping lift / rigid vang לתמיכה ב-boom
- [ ] הכינו sail ties בהישג יד
- [ ] שחררו את ה-main halyard, ודאו שינוע

## הורדה
- [ ] **פנו אל הרוח (head-to-wind) כדי להוריד כוח מהמפרש**
- [ ] שחררו mainsheet ו-vang
- [ ] הורידו את ה-halyard בהדרגה; הצוות אוסף את המפרש על ה-boom
- [ ] קפלו את המפרש מצד לצד מעל ה-boom (או הזינו ל-stackpack)

## אבטחה
- [ ] Sail ties במרווחים; צמודים אך לא מועכים battens
- [ ] **אשרו שה-halyard מאובטח ולא יונף בטעות**
- [ ] שחררו/כווננו את ה-topping lift כך שה-boom בגובה עבודה
- [ ] הרכיבו כיסוי מפרש אם סיימתם להיום

## הערות ומלכודות
- Head-to-wind מונע מהמפרש להתמלא ולצנוח למים.
- שימו לב לאצבעות בין המפרש/slides לבין מסילת התורן בזמן ההורדה.
