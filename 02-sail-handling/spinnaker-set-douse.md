# Spinnaker Set & Douse

- **Category:** Sail handling
- **Style:** READ–DO
- **When to use:** Downwind sailing in suitable conditions and crew.
- **Crew:** Helm + bow + pit/trimmer (min 3 recommended).

> Killer items are shown in **bold**. High-load sail — brief the crew first.

## Prepare to set (symmetric, from a bag/turtle)
- [ ] **Confirm conditions are within crew/boat limits before setting**
- [ ] Rig guy, sheet, halyard, and (if used) pole topping lift & downhaul
- [ ] Pack/verify the sail is not twisted; three corners accessible
- [ ] Set the pole to the forestay, square for the wind angle

## Set / hoist
1. [ ] Bear away to a broad reach / run to blanket the sail behind the main
2. [ ] Pre-set the guy so the tack goes to the pole end
3. [ ] **Hoist fast to the top before the sail fills**
4. [ ] Trim sheet; ease as needed; adjust pole height so clews are level
5. [ ] Once flying, ease the headsail / furl it away

## Douse
1. [ ] Bear away to blanket the spinnaker behind the mainsail
2. [ ] **Ease the guy / let the pole go forward to collapse the sail**
3. [ ] Gather the foot in the lee of the main; ease the halyard under control
4. [ ] Pull the sail down through the companionway / into a bag
5. [ ] **Confirm halyard clear and secured; stow pole**

## Notes & gotchas
- Blanketing behind the main is the key to a controlled set and douse.
- If it broaches or wraps, ease sheets and head to depower; don't fight the load.
- Consider a snuffer/sock or top-down furler for shorthanded crews.
