# פרישה והורדה של Spinnaker

- **קטגוריה:** טיפול במפרשים
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** שיט עם הרוח בתנאים וצוות מתאימים.
- **צוות:** הגה + חרטום + pit/trimmer (מומלץ מינימום 3).

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. מפרש בעומס גבוה — תדרכו את הצוות קודם.

## הכנה לפרישה (symmetric, משק/turtle)
- [ ] **אשרו שהתנאים בגבולות הצוות/הסירה לפני הפרישה**
- [ ] הכינו guy, sheet, halyard, ו-(אם בשימוש) pole topping lift & downhaul
- [ ] ארזו/ודאו שהמפרש אינו מפותל; שלוש הפינות נגישות
- [ ] כווננו את ה-pole ל-forestay, מרובע לזווית הרוח

## פרישה / הנפה
1. [ ] פנו עם הרוח ל-broad reach / run כדי להצל את המפרש מאחורי הראשי
2. [ ] כווננו מראש את ה-guy כך שה-tack יגיע לקצה ה-pole
3. [ ] **הניפו מהר עד למעלה לפני שהמפרש מתמלא**
4. [ ] טרמו sheet; שחררו לפי הצורך; כווננו גובה pole כך שה-clews מפולסים
5. [ ] כשעף, שחררו את מפרש החרטום / גללו אותו

## הורדה
1. [ ] פנו עם הרוח כדי להצל את ה-spinnaker מאחורי המפרש הראשי
2. [ ] **שחררו את ה-guy / תנו ל-pole לנוע קדימה כדי למוטט את המפרש**
3. [ ] אספו את ה-foot במחסה הראשי; שחררו את ה-halyard בשליטה
4. [ ] משכו את המפרש דרך ה-companionway / לתוך שק
5. [ ] **אשרו שה-halyard פנוי ומאובטח; אחסנו את ה-pole**

## הערות ומלכודות
- הצללה מאחורי הראשי היא המפתח לפרישה והורדה מבוקרות.
- אם broach או wrap, שחררו sheets ופנו כדי להוריד כוח; אל תיאבקו בעומס.
- שקלו snuffer/sock או top-down furler לצוותים מצומצמים.
