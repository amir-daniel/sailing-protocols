# קיצור המפרש הראשי (slab / single-line)

- **קטגוריה:** טיפול במפרשים
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** הרוח מתגברת; "ריף מוקדם" — ברגע שחושבים על זה לראשונה.
- **צוות:** הגה + אחד בתורן/קוקפיט.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## החליטו
- [ ] **קצרו מוקדם — לפני שזה נעשה קשה/מסוכן.** אם בספק, קצרו.
- [ ] בחרו את מספר ה-reef לרוח הצפויה

## הכנה
- [ ] פנו אל הרוח כדי להפחית עומס; שחררו mainsheet ו-vang
- [ ] תפסו את ה-topping lift (או העמידו את ה-boom על rigid vang)
- [ ] ודאו ש-reefing lines חופשיים לנוע

## בצעו את ה-reef
1. [ ] שחררו את ה-main halyard לנקודה המסומנת ל-reef זה
2. [ ] **חברו / אבטחו את luff reef cringle ב-tack (gooseneck)**
3. [ ] מתחו מחדש וקבעו את ה-main halyard
4. [ ] משכו את leech reef line עד ש-clew cringle יורד ל-boom
5. [ ] קבעו/משכו את ה-reef line בחוזקה
6. [ ] שחררו בחזרה את ה-topping lift

## אשרו וסדרו
- [ ] **אשרו ש-tack וגם clew מאובטחים לפני העמסת המפרש**
- [ ] טרמו mainsheet/vang; פנו עם הרוח למסלול
- [ ] סדרו מפרש רופף עם reef ties (אל תקשרו ל-boom תחת עומס)
- [ ] כרכו ואחסנו קצוות

## הערות ומלכודות
- קודם tack, אחר כך clew — clew בעומס עם tack לא מאובטח עלול לקרוע את המפרש.
- Reef ties מיועדים לסדר בלבד; ה-tack וה-clew נושאים את העומס.
