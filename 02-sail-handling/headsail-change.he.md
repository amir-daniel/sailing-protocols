# החלפת מפרש חרטום (hank-on / twin-groove foil)

- **קטגוריה:** טיפול במפרשים
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** מעבר למפרש חרטום קטן/גדול יותר כשהרוח משתנה.
- **צוות:** הגה + אחד או שניים ב-foredeck.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. עבודה ב-foredeck — התחברו בחבל.

## הכנה
- [ ] **הצוות ב-foredeck מחובר ל-jackstays**
- [ ] הביאו את המפרש החדש קדימה בשקו, קשור לסירה
- [ ] הביאו את הסירה ל-reach ל-foredeck יציב ושטוח יותר

## החלפה (שיטת bald-headed, hank-on)
1. [ ] חברו (hank on) את המפרש החדש מתחת ל-hank התחתון של הישן (אם יש מקום) או הכינו בצד
2. [ ] הורידו את המפרש הישן; **אבטחו אותו לסיפון מיד כדי שלא ייפול למים**
3. [ ] נתקו (unhank) את הישן; חברו את החדש מה-tack ומעלה, בסדר
4. [ ] העבירו את ה-halyard וה-sheets למפרש החדש
5. [ ] **אשרו שה-hanks בסדר ולא מוצלבים; tack shackle נעול**

## הנפה וטרים
- [ ] הניפו את המפרש החדש; מתחו halyard
- [ ] כווננו מיקום sheet-car לגודל המפרש החדש
- [ ] פנו עם הרוח, טרמו, חזרו למסלול
- [ ] ארזו ואחסנו את המפרש הישן למטה

## הערות ומלכודות
- לעולם אל תתנו למפרש שהורד לתלות מעבר לדופן — הוא עלול להתמלא במים
  ולגרור מישהו או להזיק לסירה.
- Twin-groove foils מאפשרים הנפה-לפני-הורדה; היזהרו משני luffs נתקעים.
