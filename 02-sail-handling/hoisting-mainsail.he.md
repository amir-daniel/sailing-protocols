# הנפת המפרש הראשי

- **קטגוריה:** טיפול במפרשים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** הרמת המפרש הראשי לאחר יציאה מהנמל / מים פתוחים.
- **צוות:** הגה + אחד ב-halyard (אפשרי לבד עם autopilot).

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הכנה
- [ ] קשירות מפרש / כיסוי הוסרו; המפרש חופשי לנוע
- [ ] Halyard מחובר ל-headboard; **לא מסובך ולא כרוך סביב דבר**
- [ ] Mainsheet ו-vang משוחררים; topping lift מכוון (אם מותקן)
- [ ] Battens/lazyjacks פנויים; reef lines מנותבים וחופשיים
- [ ] Boom מאובטח עד שמוכנים להנפה

## הנפה
- [ ] **הפנו את הסירה אל מול הרוח (head-to-wind, או קרוב) כדי להוריד כוח מהמפרש**
- [ ] שחררו את ה-sheet לחלוטין כדי שהמפרש יתנופף
- [ ] הניפו בזריזות; שימו לב שה-battens לא נתקעים ב-lazyjacks
- [ ] משכו ב-winch את החלק האחרון למתח luff מלא
- [ ] **אשרו שה-halyard למעלה לגמרי ומקובע/נעול**

## טרים ואישור
- [ ] שחררו/כווננו את ה-topping lift כך שלא ייקח עומס
- [ ] פנו עם הרוח למסלול; טרמו mainsheet ו-vang
- [ ] כווננו מתח luff (Cunningham) לתנאים
- [ ] סדרו וכרכו את קצה ה-halyard

## הערות ומלכודות
- Head-to-wind מפחית עומס והיתקעות battens; אל תניפו כשהמפרש בעומס.
- אם ה-halyard לא עולה עד הסוף, בדקו batten תקוע ב-lazyjacks
  לפני משיכה חזקה ב-winch.
