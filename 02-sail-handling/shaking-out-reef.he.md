# שחרור reef (Shaking Out)

- **קטגוריה:** טיפול במפרשים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** הרוח נחלשה והתייצבה; מוכנים לשאת יותר מפרש.
- **צוות:** הגה + אחד בתורן/קוקפיט.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## החליטו
- [ ] אשרו שהרגיעה יציבה, לא הפוגה זמנית
- [ ] שחררו reef אחד בכל פעם

## הכנה
- [ ] פנו אל הרוח כדי להפחית עומס; שחררו mainsheet ו-vang
- [ ] תפסו את ה-topping lift (או העמידו rigid vang)
- [ ] התירו כל reef ties במפרש **(את כולם — שאף אחד לא נשאר קשור)**

## שחרור
1. [ ] שחררו את leech reef line
2. [ ] שחררו מעט את ה-main halyard כדי לפנות את ה-tack cringle
3. [ ] **נתקו / שחררו את luff tack cringle ב-gooseneck**
4. [ ] הניפו את ה-main halyard בחזרה לגובה מלא; מתחו את ה-luff
5. [ ] קבעו/נעלו את ה-halyard

## אשרו וסדרו
- [ ] **אשרו שלא נותרו reef ties במפרש לפני העמסתו**
- [ ] שחררו בחזרה את ה-topping lift
- [ ] טרמו, פנו עם הרוח למסלול
- [ ] סדרו וכרכו קצוות reef line ו-halyard

## הערות ומלכודות
- reef tie שנשכח קשור יקרע את המפרש כשיתמלא — בדקו את כל ה-foot לפני ההנפה.
