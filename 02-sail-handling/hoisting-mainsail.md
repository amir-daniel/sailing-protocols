# Hoisting the Mainsail

- **Category:** Sail handling
- **Style:** DO–CONFIRM
- **When to use:** Setting the main after leaving harbour / open water.
- **Crew:** Helm + one on the halyard (can be single-handed with autopilot).

> Killer items are shown in **bold**.

## Prepare
- [ ] Sail ties / cover removed; sail free to run
- [ ] Halyard attached to headboard; **not fouled or wrapped around anything**
- [ ] Mainsheet and vang eased; topping lift set (if fitted)
- [ ] Battens/lazyjacks clear; reef lines run and free
- [ ] Boom secured until ready to hoist

## Hoist
- [ ] **Turn boat head-to-wind (or close) to depower the sail**
- [ ] Ease sheet fully so sail can flag
- [ ] Hoist smartly; watch battens don't snag lazyjacks
- [ ] Winch the last section to full luff tension
- [ ] **Confirm halyard is fully up and cleated/locked**

## Trim & confirm
- [ ] Ease/re-set topping lift so it's not taking load
- [ ] Bear away onto course; trim mainsheet and vang
- [ ] Adjust luff tension (Cunningham) for conditions
- [ ] Tidy and coil the halyard tail

## Notes & gotchas
- Head-to-wind reduces load and batten snags; don't hoist with the sail loaded.
- If the halyard won't run to the top, check for a batten caught in lazyjacks
  before winching hard.
