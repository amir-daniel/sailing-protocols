# טרום-התנעה והתנעת המנוע

- **קטגוריה:** מערכות ומנוע
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** לפני התנעת הדיזל (בכל פעם).
- **צוות:** סקיפר או מהנדס.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. זכרו **WOBBLE** כעזר זיכרון.

## בדיקות טרום-התנעה (WOBBLE)
- [ ] **W**ater (מים) — רמת נוזל קירור תקינה; **raw-water seacock OPEN**
- [ ] **O**il (שמן) — רמת שמן המנוע ב-dipstick; שמן תיבת הילוכים
- [ ] **B**elts (רצועות) — מתח ומצב רצועת ה-alternator/משאבת מים
- [ ] **B**ilge (תחתית) — יבשה; אין הצטברות שמן/דלק/מים
- [ ] **L**ooks (בדיקה חזותית) — אין דליפות, טפטופים, חלקים רופפים, שחיקה
- [ ] **E**xhaust / **E**lectrics (פליטה/חשמל) — battery isolator דולק; פליטה פנויה

## התנעה
1. [ ] ידית הילוכים ב-**neutral**
2. [ ] הדלקה; בדקו שנורות אזהרה/התרעות נדלקות
3. [ ] התניעו את המנוע
4. [ ] **אשרו שמי קירור זורמים מהפליטה תוך ~10–15 שנ'**
5. [ ] בדקו שנורות אזהרת לחץ שמן וטעינה נכבות
6. [ ] הקשיבו לרעש/רטט חריג; בדקו שאין התרעות

## חימום ואישור
- [ ] תנו לו להתחמם בסל"ד נמוך לפני העמסה
- [ ] בדקו הילוכים קדימה ואחורה בעודכם קשורים (או עם מים פנויים)
- [ ] בדקו שהטמפרטורה מתייצבת בטווח הנורמלי

## הערות ומלכודות
- **אין מי קירור בפליטה? עצרו מיד** — סביר seacock סגור,
  intake חסום, או impeller שכשל. ריצה יבשה הורסת את ה-impeller ועלולה
  לגרום להתחממות יתר במהירות.
