# הפעלת Heads (שירותים) וחסימה

- **קטגוריה:** מערכות ומנוע
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** תדרוך אורחים; פתיחת חסימה בשירותים ימיים.
- **צוות:** כל אחד; סקיפר לחסימות.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**. "אם לא אכלת את זה, אל תשטוף את זה."

## הפעלה רגילה (manual pump heads)
- [ ] Inlet/flush seacock **פתוח**, outlet seacock **פתוח** לפני שימוש
- [ ] ידית ל-"flush" (רטוב): שאבו כדי להוסיף מים ולנקות את הקערה
- [ ] ידית ל-"dry": שאבו כדי לרוקן את הקערה בלי להוסיף מים
- [ ] **שום דבר לא נכנס שלא נאכל קודם** — אין מגבונים, פריטי היגיינה, נייר עודף
- [ ] השאירו את הקערה ריקה (יבשה) ושאובה לאחר שימוש

## עזיבת הסירה / במרחוק במזג אוויר גס
- [ ] **סגרו את ה-inlet ו-outlet seacocks** (מונע back-flooding/siphoning)
- [ ] תדרכו את הצוות שיש לפתוח מחדש את ה-seacocks לפני השימוש הבא

## פתיחת חסימה
1. [ ] הפסיקו לשאוב נגד התנגדות — **אל תכריחו** (תפוצצו צינור/מחבר)
2. [ ] בדקו ש-**outlet seacock באמת פתוח** (הגורם מספר 1)
3. [ ] בדקו שה-vented loop / anti-siphon valve אינו סתום
4. [ ] פרקו את המשאבה; פתחו את החסימה (כפפות; צפו לבלגן)
5. [ ] בדקו ב-outlet hose **הצטברות סידן** (גורם נפוץ לטווח ארוך)

## הערות ומלכודות
- רוב ה"חסימות" הן פשוט outlet seacock סגור — בדקו זאת קודם.
- כפייה על משאבה חסומה מפצלת צינורות/מחברים ועלולה לגרום להצפה איטית — עצרו ואבחנו.
