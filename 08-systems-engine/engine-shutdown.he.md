# כיבוי המנוע

- **קטגוריה:** מערכות ומנוע
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** עצירת המנוע לאחר שימוש.
- **צוות:** סקיפר או מהנדס.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## לפני העצירה
- [ ] ידית הילוכים ב-**neutral**
- [ ] תנו למנוע לפעול בסרק דקה-שתיים כדי להתקרר (במיוחד אחרי ריצה קשה / turbo)

## עצירה
1. [ ] הפעילו את בקרת העצירה עד שהמנוע נעצר
2. [ ] כבו את ההדלקה; **אשרו שהבאזר של ההתרעה נפסק** (מפתח כבוי לחלוטין)
3. [ ] **שקלו לסגור את ה-raw-water seacock** אם עוזבים את הסירה / מזג אוויר גס (למניעת siphoning / הצפה)

## לאחר מכן
- [ ] כבו מצבר/isolator המנוע אם עוזבים את הסירה (לפי ההגדרה שלכם)
- [ ] רשמו שעות ריצה ביומן
- [ ] בדיקה חזותית מהירה: דליפות, טפטופים או אבק רצועה חדשים?

## הערות ומלכודות
- השארת ההדלקה דולקת חלקית מרוקנת את המצבר ועלולה להזיק ל-alternator — אשרו שהיא כבויה לחלוטין.
- אם סגרתם את ה-raw-water seacock, **תייגו את ההגה/מפתח כדי שאף אחד לא יתניע יבש**.
