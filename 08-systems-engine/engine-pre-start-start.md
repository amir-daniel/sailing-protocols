# Engine Pre-Start & Start

- **Category:** Systems & engine
- **Style:** DO–CONFIRM
- **When to use:** Before starting the diesel (every time).
- **Crew:** Skipper or engineer.

> Killer items are shown in **bold**. Remember **WOBBLE** as a memory aid.

## Pre-start checks (WOBBLE)
- [ ] **W**ater — coolant level OK; **raw-water seacock OPEN**
- [ ] **O**il — engine oil level on the dipstick; gearbox oil
- [ ] **B**elts — alternator/water-pump belt tension and condition
- [ ] **B**ilge — dry; no oil/fuel/water pooling
- [ ] **L**ooks — visual: no leaks, drips, loose parts, chafe
- [ ] **E**xhaust / **E**lectrics — battery isolator on; exhaust clear

## Start
1. [ ] Gear lever in **neutral**
2. [ ] Ignition on; check warning lights/alarms come on
3. [ ] Start the engine
4. [ ] **Confirm cooling water is flowing out of the exhaust within ~10–15s**
5. [ ] Check oil-pressure and charge warning lights go OUT
6. [ ] Listen for unusual noise/vibration; check no alarms

## Warm-up & confirm
- [ ] Let it warm at low revs before loading
- [ ] Test ahead and astern gears while still tied up (or with clear water)
- [ ] Check the temperature settles in the normal range

## Notes & gotchas
- **No cooling water at the exhaust? Stop immediately** — likely closed seacock,
  blocked intake, or failed impeller. Running dry destroys the impeller and can
  cause overheating fast.
