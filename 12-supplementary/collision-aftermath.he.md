# התנגשות (אחרי האירוע)

- **קטגוריה:** משלים
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** מיד לאחר התנגשות עם כלי אחר או עצם.
- **צוות:** כל הידיים.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## מיידי
1. [ ] **בדקו פציעות בכל הצוות** (שלכם, ואם אפשר גם של הכלי האחר)
2. [ ] **בדקו חדירת מים / נזק לגוף** (ראו רשימת flooding)
3. [ ] בדקו שריפה/דליפת דלק; בדקו שהכלונסאות עומדות
4. [ ] עצרו את הסירה / הורידו התנע אם בטוח; העריכו לפני תמרון נוסף

## בטיחות חיים קודם
- [ ] הגישו סיוע לכלי האחר אם בטוח — **אתם מחויבים חוקית** לסייע
      לאנשים בסכנה אם ניתן ללא סיכון חמור לכלי שלכם
- [ ] אם מישהו בסכנה: **Mayday / PAN-PAN** לפי העניין
- [ ] טפלו בפציעות (ראו חירום רפואי)

## נהלו את הכלי
- [ ] טפלו בהצפה/נזק; הפעילו משאבות; פקקו/חזקו פרצות
- [ ] אם כשיר שיט, המשיכו לנמל בטוח; אם לא, היערכו בהתאם
- [ ] שמרו על הכלונסאות אם ניזוקו (ראו dismasting)

## אדמיניסטרציה (כשבטוח)
- [ ] **החליפו פרטים** עם הכלי האחר (שם, בעלים, מבטח, איש קשר)
- [ ] **תעדו הכול:** זמן, מיקום, תנאים, מה קרה, תמונות
- [ ] ציינו עדים
- [ ] דווחו לרשות/משמר החופים/המבטח הרלוונטיים כנדרש

## הערות ומלכודות
- בטיחות האנשים והכלי קודמת; הניירת מגיעה אחרי שכולם בטוחים.
- קיימת חובה חוקית להגיש סיוע לאחר התנגשות כאשר ניתן לעשות זאת בבטחה.
