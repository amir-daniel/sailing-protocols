# הפעלת סירת עזר (Dinghy / Tender)

- **קטגוריה:** משלים
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** שימוש בסירת העזר להגעה לחוף / בין הסירה לחוף.
- **צוות:** כל אחד.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## לפני שיוצאים
- [ ] **אפודי הצלה על כולם** (סירת העזר היא המקום שבו קורות טביעות רבות)
- [ ] אל **תעמיסו יתר על המידה** — כבדו את מגבלת האנשים/המשקל
- [ ] Bailer, painter, ו-(במרחוק/בלילה) פנס ו-VHF/טלפון ידני על הסירה
- [ ] משוטים על הסירה גם אם משתמשים במנוע חיצוני (הנעת גיבוי)
- [ ] Kill-cord מחובר לנהג (אם מותקן מנוע חיצוני)

## עלייה ותנועה
- [ ] עלו אחד בכל פעם, דורכים ל**מרכז**; הישארו נמוך
- [ ] פזרו משקל באופן שווה; שמרו את הצוות יושב
- [ ] נועו במהירות בטוחה; שימו לב לשוחים, חבלים וכלים אחרים
- [ ] דעו את מצב הדלק והטווח; קחו בחשבון רוח/זרם שדוחפים אתכם מהמסלול

## חזרה / אחסון
- [ ] התקרבו לסירה/רציף נגד הרוח/הגאות; **אבטחו את ה-painter**
- [ ] הרימו/גררו/אחסנו את סירת העזר בבטחה; **הסירו את הפקק (bung) באחסון** (אם רלוונטי)
- [ ] נעלו או אבטחו את המנוע החיצוני וסירת העזר מפני גניבה

## הערות ומלכודות
- מים קרים והעמסת יתר הופכים את סירת העזר לסירה הקטנה המסוכנת ביותר שתשתמשו בה —
  אפודי הצלה תמיד, ללא יוצא מן הכלל, גם לחתירה קצרה.
- סירת עזר שנסחפת ובורחת בזרם היא חירום נפוץ (ונמנע) — אבטחו את ה-painter.
