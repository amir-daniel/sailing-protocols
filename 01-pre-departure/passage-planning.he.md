# תכנון מעבר (Passage Planning)

- **קטגוריה:** טרום יציאה
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** לפני כל מעבר, רצוי יום לפני היציאה.
- **צוות:** סקיפר (עם נווט אם זמין).

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## מזג אוויר וסביבה
- [ ] השיגו תחזית עדכנית (GRIB, חופית, ספנות) לכל חלון הזמן
- [ ] **זהו אזהרות סערה/רוח חזקה או מגמות לא נוחות**
- [ ] שימו לב לכיוון/עוצמת הרוח מול המסלול המתוכנן (beat/reach/run)
- [ ] בדקו גאות וזרמי גאות ליציאה, ל"שערים" מרכזיים ולהגעה
- [ ] אשרו שעות אור; זהו קטעי מעבר לילי

## מסלול
- [ ] שרטטו מסלול עם waypoints; ציינו מרחק כולל
- [ ] העריכו מהירות סירה / ETA וטווח דלק מול מרחק
- [ ] **זהו מכשולים: שרטונות, סלעים, TSS, אזורים מוגבלים, overfalls**
- [ ] סמנו נמלי מפלט לאורך המסלול
- [ ] ציינו "שערי" גאות קריטיים וחלון הזמן לעמוד בהם

## הגעה
- [ ] בדקו את נמל היעד: כניסה, עומקים, שעות, איש קשר/VHF
- [ ] אשרו זמינות מעגן/עוגן ואחיזה
- [ ] זהו סימני ניווט/הובלה לכניסה

## אדמיניסטרציה
- [ ] הגישו תוכנית מעבר / float plan לאיש קשר ביבשה
- [ ] אשרו זמינות צוות וכל צורך במכס/הגירה
- [ ] מפות (נייר + אלקטרוני) על הסירה ומעודכנות

## הערות ומלכודות
- תכננו את ההגעה לשעות אור וגאות נוחה כשאפשר.
- החזיקו תמיד תוכנית ב' אם חלון מזג האוויר נסגר.
