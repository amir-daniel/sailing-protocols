# בדיקת סירה ובטיחות

- **קטגוריה:** טרום יציאה
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** לפני עזיבת הרציף/המעגן, בכל פעם.
- **צוות:** סקיפר; האצילו חלקים לצוות.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## מתחת לקו המים / מנוע
- [ ] **Seacocks במצב הנכון; אין דליפות ב-through-hulls**
- [ ] Bilge יבש / משאבות bilge פועלות (ידנית + חשמלית)
- [ ] רמות שמן ונוזל קירור של המנוע נבדקו
- [ ] רמת דלק מספקת למעבר + רזרבה
- [ ] **Raw-water intake פתוח; זרימת מי קירור מאושרת לאחר התנעה**
- [ ] Stern gland / stuffing box תקין

## סיפון וכלונסאות
- [ ] הכלונסאות נבדקו חזותית (shrouds, stays, spreaders, אביזרים)
- [ ] Halyards ו-sheets מנותבים נכון, ללא שחיקה, לא מסובכים
- [ ] ציוד ה-reefing מוכן לפעולה
- [ ] ידיות winch על הסירה ומאוחסנות
- [ ] עוגן ו-rode מוכנים, מאובטחים אך נגישים

## ציוד בטיחות
- [ ] **אפודי הצלה חבושים, מידה נכונה, גז/אור תקין; רתמות ו-tethers מוכנים**
- [ ] Jackstays מוקמו למרחוק/לילה/מזג אוויר קשה
- [ ] Liferaft בתוקף ומאובטח; grab bag מוכן (למרחוק)
- [ ] רקטות בתוקף; המיקום ידוע
- [ ] מטפים ושמיכת אש במקומם ובתוקף
- [ ] ערכת עזרה ראשונה על הסירה ונגישה
- [ ] Danbuoy / מצוף פרסה / ציוד MOB מוכן ומורכב

## אלקטרוניקה ותקשורת
- [ ] VHF פועל; **DSC/MMSI מתוכנת; בדיקת קשר בוצעה**
- [ ] אורות ניווט פועלים
- [ ] Chartplotter / GPS דולק ופועל
- [ ] EPIRB / PLB רשומים, בתוקף, נגישים
- [ ] טלפון טעון; ניווט גיבוי זמין

## הערות ומלכודות
- אשרו את סימן מי הקירור *אחרי* שהמנוע מותנע, לא לפני.
- אם פריט קריטי כלשהו נכשל, אל תצאו עד שנפתר.
