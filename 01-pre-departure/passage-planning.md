# Passage Planning

- **Category:** Pre-departure
- **Style:** READ–DO
- **When to use:** Before any passage, ideally the day before departure.
- **Crew:** Skipper (with navigator if available).

> Killer items are shown in **bold**.

## Weather & environment
- [ ] Obtain latest forecast (GRIB, coastal, shipping) for the whole window
- [ ] **Identify any gale/storm warnings or unfavourable trends**
- [ ] Note wind direction/strength vs. planned course (beat/reach/run)
- [ ] Check tides & tidal streams for departure, key gates, and arrival
- [ ] Confirm daylight hours; identify night passage segments

## Route
- [ ] Plot route with waypoints; note total distance
- [ ] Estimate boat speed / ETA and fuel range vs. distance
- [ ] **Identify hazards: shoals, rocks, TSS, restricted areas, overfalls**
- [ ] Mark bail-out ports / ports of refuge along the route
- [ ] Note critical tidal gates and the time window to make them

## Arrival
- [ ] Check destination harbour: entrance, depths, hours, contact/VHF
- [ ] Confirm berth/anchorage availability and holding
- [ ] Identify pilotage/leading marks for entry

## Admin
- [ ] File a passage plan / float plan with a shore contact
- [ ] Confirm crew availability and any customs/immigration needs
- [ ] Charts (paper + electronic) aboard and up to date

## Notes & gotchas
- Plan the arrival for daylight and favourable tide where possible.
- Always have a plan B if the weather window closes.
