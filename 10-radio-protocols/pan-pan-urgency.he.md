# PAN-PAN (דחיפות)

- **קטגוריה:** פרוטוקולי קשר
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** הודעה **דחופה** בנוגע לבטיחות כלי שיט או אדם,
  אך **לא** סכנה חמורה ומיידית (למשל כשל מנוע ללא סכנה מיידית,
  פציעה קלה הזקוקה לייעוץ, מצב מדאיג אך יציב).
- **ציוד:** VHF Ch 16 (DSC urgency אם מותקן).

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## התרעת DSC urgency (אופציונלי)
- [ ] שדרו קריאת **DSC urgency** אם המכשיר תומך, ואז קול ב-Ch 16

## קריאת קול ב-Ch 16
```
PAN-PAN, PAN-PAN, PAN-PAN
ALL STATIONS, ALL STATIONS, ALL STATIONS   (or "HELLO ALL STATIONS x3")
THIS IS <boat name, three times> <callsign / MMSI>
MY POSITION IS <lat/long OR bearing & distance from a known point>
<Nature of the urgency — e.g. "engine failure, drifting, no immediate danger">
<Assistance required / intentions>
<Number of persons on board, if relevant>
OVER
```

## ייעוץ רפואי (PAN-PAN medico)
- [ ] קראו למשמר החופים; בקשו "**medical advice**" (חינם) או "**medical assistance**" (פינוי)
- [ ] מסרו את מצב המטופל, סימנים חיוניים, גיל, היסטוריה ואת מיקומכם

## לאחר מכן
- [ ] **הקשיבו לתשובה; חזרו אם אין מענה**
- [ ] פעלו לפי הוראות משמר החופים/תחנות אחרות
- [ ] **שדרגו ל-MAYDAY מיד אם המצב הופך חמור**
- [ ] בטלו את ה-PAN-PAN כשנפתר:
```
PAN-PAN, ALL STATIONS x3, THIS IS <boat name x3>, <time>, PAN-PAN <time> CANCELLED. OUT
```

## הערות ומלכודות
- PAN-PAN נהגה "PAN PAN" (מצרפתית *panne* = תקלה).
- אם מתלבטים בין PAN-PAN ל-Mayday וחיים בסכנה — השתמשו ב-Mayday.
