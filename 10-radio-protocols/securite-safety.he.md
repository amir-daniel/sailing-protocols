# Sécurité (בטיחות)

- **קטגוריה:** פרוטוקולי קשר
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** הודעת **בטיחות** — אזהרות ניווט או מטאורולוגיות
  (למשל מכשול שנצפה, כלי שיט שאינו בשליטה, עצם בנתיב השיט,
  אזהרת רוח חזקה). אין סכנה לכלי שלכם.
- **ציוד:** VHF Ch 16 להכרזה, ואז ערוץ עבודה למסירת ההודעה.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## הכריזו ב-Ch 16
```
SÉCURITÉ, SÉCURITÉ, SÉCURITÉ
ALL STATIONS, ALL STATIONS, ALL STATIONS
THIS IS <boat name, three times> <callsign / MMSI>
<Announce the message and the working channel to switch to>
e.g. "FOR NAVIGATION WARNING, LISTEN CHANNEL SIX SEVEN"
OUT
```

## מסרו את ההודעה בערוץ העבודה
- [ ] עברו לערוץ העבודה שהוכרז (למשל Ch 67)
- [ ] מסרו את מידע הבטיחות בבירור:
```
SÉCURITÉ, SÉCURITÉ, SÉCURITÉ
THIS IS <boat name x3>
<The safety message — position and description of the hazard>
OUT
```

## הערות ומלכודות
- נהגה "say-cure-ee-tay" (מצרפתית *sécurité* = בטיחות).
- משמר החופים משדר באופן שגרתי הודעות Sécurité עבור אזהרות סערה ואזהרות
  ניווט — מוכרז ב-16, נמסר בערוץ עבודה.
- השתמשו עבור דברים כמו מכולה צפה, לווייתן, שבר ספינה, מצוף לא מואר.
