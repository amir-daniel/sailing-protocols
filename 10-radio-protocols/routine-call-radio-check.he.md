# קריאת VHF שגרתית ובדיקת קשר

- **קטגוריה:** פרוטוקולי קשר
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** קריאות שגרתיות לתחנה אחרת; בדיקת המכשיר.
- **ציוד:** VHF Ch 16 (קריאה), ואז ערוץ עבודה.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## קריאה שגרתית לתחנה אחרת
1. [ ] הקשיבו תחילה — **אל תשדרו על גבי שיחה קיימת**
2. [ ] קראו ב-Ch 16 (או בערוץ העבודה/הקריאה של התחנה):
```
<Station being called, up to 3 times>
THIS IS <your boat name, up to 3 times> <callsign>
OVER
```
3. [ ] כשעונים, **הסכימו ועברו לערוץ עבודה** (לא 16):
```
<Station>, THIS IS <boat name>, CHANNEL <xx>, OVER
```
4. [ ] נהלו את השיחה בערוץ העבודה; סיימו ב-"OUT"

## בדיקת קשר (Radio check)
- [ ] **אל תשתמשו ב-Ch 16 לבדיקות קשר**
- [ ] השתמשו בערוץ עבודה, או קראו למרינה/**למשמר החופים בערוץ עבודה**
- [ ] תסריט:
```
<Station, or "ANY STATION"> RADIO CHECK
THIS IS <boat name x3> <callsign>
HOW DO YOU READ ME? OVER
```
- [ ] דווחו על איכות האות בסולם קריאוּת 1–5 (למשל "**you are loud and clear**" / "readability 5")

## Prowords (השתמשו במונחים תקניים ופשוטים)
- **OVER** — סיימתי, מצפה לתשובה
- **OUT** — השיחה הסתיימה (לעולם לא "over and out")
- **ROGER** — התקבל והובן
- **SAY AGAIN** — חזור על הודעתך
- **WILCO** — אמלא (will comply)
- **STATION CALLING** — כשלא קלטתם מי קרא

## הערות ומלכודות
- Ch 16 מיועד ל**מצוקה, דחיפות, בטיחות וקריאה ראשונית בלבד** — העבירו
  שיחת חולין לערוץ עבודה במהירות.
- דברו בבירור, בקצב הכתבה; השתמשו באלפבית הפונטי לשמות/callsigns.
