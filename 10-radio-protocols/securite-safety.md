# Sécurité (Safety)

- **Category:** Radio protocols
- **Style:** READ–DO
- **When to use:** A **safety** message — navigational or meteorological warnings
  (e.g. a hazard sighted, a vessel not under command, an object in the fairway,
  a strong-wind warning). No danger to your own vessel.
- **Equipment:** VHF Ch 16 to announce, then a working channel to pass the message.

> Killer items are shown in **bold**.

## Announce on Ch 16
```
SÉCURITÉ, SÉCURITÉ, SÉCURITÉ
ALL STATIONS, ALL STATIONS, ALL STATIONS
THIS IS <boat name, three times> <callsign / MMSI>
<Announce the message and the working channel to switch to>
e.g. "FOR NAVIGATION WARNING, LISTEN CHANNEL SIX SEVEN"
OUT
```

## Pass the message on the working channel
- [ ] Switch to the announced working channel (e.g. Ch 67)
- [ ] Give the safety information clearly:
```
SÉCURITÉ, SÉCURITÉ, SÉCURITÉ
THIS IS <boat name x3>
<The safety message — position and description of the hazard>
OUT
```

## Notes & gotchas
- Spoken "say-cure-ee-tay" (from French *sécurité* = safety).
- Coastguards routinely broadcast Sécurité messages for gale warnings and nav
  warnings — announced on 16, passed on a working channel.
- Use for things like a floating container, a whale, a wreck, an unlit buoy.
