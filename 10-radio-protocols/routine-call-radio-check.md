# Routine VHF Call & Radio Check

- **Category:** Radio protocols
- **Style:** DO–CONFIRM
- **When to use:** Routine calls to another station; testing the radio.
- **Equipment:** VHF Ch 16 (calling), then a working channel.

> Killer items are shown in **bold**.

## Routine call to another station
1. [ ] Listen first — **don't transmit over an existing conversation**
2. [ ] Call on Ch 16 (or the station's working/calling channel):
```
<Station being called, up to 3 times>
THIS IS <your boat name, up to 3 times> <callsign>
OVER
```
3. [ ] When answered, **agree and switch to a working channel** (not 16):
```
<Station>, THIS IS <boat name>, CHANNEL <xx>, OVER
```
4. [ ] Conduct the conversation on the working channel; end with "OUT"

## Radio check
- [ ] **Do NOT use Ch 16 for radio checks**
- [ ] Use a working channel, or call a marina/**a Coastguard on a working channel**
- [ ] Script:
```
<Station, or "ANY STATION"> RADIO CHECK
THIS IS <boat name x3> <callsign>
HOW DO YOU READ ME? OVER
```
- [ ] Report signal quality by readability 1–5 (e.g. "**you are loud and clear**" / "readability 5")

## Prowords (use plain, standard terms)
- **OVER** — I've finished, expecting a reply
- **OUT** — conversation ended (never "over and out")
- **ROGER** — received and understood
- **SAY AGAIN** — repeat your message
- **WILCO** — will comply
- **STATION CALLING** — used when you didn't catch who called

## Notes & gotchas
- Ch 16 is for **distress, urgency, safety, and initial calling only** — move
  routine chat to a working channel promptly.
- Speak clearly, at dictation speed; use the phonetic alphabet for names/callsigns.
