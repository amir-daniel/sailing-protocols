# Mayday (Distress) — Grave & Imminent Danger

- **Category:** Radio protocols
- **Style:** READ–DO
- **When to use:** **Grave and imminent danger** to a vessel or person, requiring
  immediate assistance (sinking, fire, MOB not recovered, serious injury).
- **Equipment:** VHF Ch 16 (and DSC distress button).

> Killer items are shown in **bold**. Read the script — don't rely on memory in a crisis.

## Step 1 — DSC alert (if fitted)
- [ ] **Lift the guard and press/hold the red DISTRESS button** (~5 seconds)
- [ ] Select nature of distress if time allows; otherwise send undesignated
- [ ] The radio auto-switches to Ch 16 — then make the voice call below

## Step 2 — Voice call on Ch 16 (high power, 25W)
Read this script aloud:

```
MAYDAY, MAYDAY, MAYDAY
THIS IS <boat name, spoken three times> <callsign / MMSI>
MAYDAY <boat name once>
MY POSITION IS <lat/long OR bearing & distance from a known point>
<Nature of distress — e.g. "we are sinking / on fire">
I REQUIRE IMMEDIATE ASSISTANCE
<Number of persons on board>
<Any other information — e.g. abandoning to liferaft, casualties>
OVER
```

## Step 3 — After transmitting
- [ ] **Release the transmit key and listen for a reply**
- [ ] If no acknowledgement, **repeat the Mayday call**
- [ ] Keep a listening watch on Ch 16; follow Coastguard instructions
- [ ] Activate EPIRB/PLB if abandoning or out of VHF range

## Relaying for another vessel (Mayday Relay)
```
MAYDAY RELAY, MAYDAY RELAY, MAYDAY RELAY
THIS IS <your boat name, three times>
<Then pass on the distress details you received>
```

## Notes & gotchas
- **P**osition, **N**ature, **N**umber on board — the three things rescuers need most.
- Send DSC first (it carries your ID and position digitally), then talk on Ch 16.
- Cancelling a false alarm is far better than staying silent — always correct it.
