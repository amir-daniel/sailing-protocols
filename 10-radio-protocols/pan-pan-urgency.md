# PAN-PAN (Urgency)

- **Category:** Radio protocols
- **Style:** READ–DO
- **When to use:** An **urgent** message about the safety of a vessel or person,
  but **NOT** grave and imminent danger (e.g. engine failure with no immediate
  danger, a minor injury needing advice, a concerning but stable situation).
- **Equipment:** VHF Ch 16 (DSC urgency if fitted).

> Killer items are shown in **bold**.

## Optional DSC urgency alert
- [ ] Send a **DSC urgency** call if your set supports it, then voice on Ch 16

## Voice call on Ch 16
```
PAN-PAN, PAN-PAN, PAN-PAN
ALL STATIONS, ALL STATIONS, ALL STATIONS   (or "HELLO ALL STATIONS x3")
THIS IS <boat name, three times> <callsign / MMSI>
MY POSITION IS <lat/long OR bearing & distance from a known point>
<Nature of the urgency — e.g. "engine failure, drifting, no immediate danger">
<Assistance required / intentions>
<Number of persons on board, if relevant>
OVER
```

## Medical advice (PAN-PAN medico)
- [ ] Call the Coastguard; request "**medical advice**" (free) or "**medical assistance**" (evacuation)
- [ ] Give patient's condition, vitals, age, history, and your position

## After
- [ ] **Listen for a reply; repeat if not answered**
- [ ] Follow Coastguard/other stations' instructions
- [ ] **Upgrade to a MAYDAY immediately if the situation becomes grave**
- [ ] Cancel the PAN-PAN when resolved:
```
PAN-PAN, ALL STATIONS x3, THIS IS <boat name x3>, <time>, PAN-PAN <time> CANCELLED. OUT
```

## Notes & gotchas
- PAN-PAN is spoken as "PAN PAN" (from French *panne* = breakdown).
- If in doubt between PAN-PAN and Mayday and life is at risk — use Mayday.
