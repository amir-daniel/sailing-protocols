# DSC — Digital Selective Calling

- **קטגוריה:** פרוטוקולי קשר
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** מדריך לשימוש ב-DSC במכשיר VHF/DSC קבוע.
- **ציוד:** VHF תומך-DSC עם **MMSI רשום** וכניסת GPS.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## דרישות מקדימות להגדרה
- [ ] **MMSI מתוכנת במכשיר ורשום** אצל הרשות
- [ ] **GPS מחובר** למכשיר (כדי שקריאות מצוקה יישאו מיקום)
- [ ] הצוות תודרך היכן כפתור ה-DSC distress וכיצד הוא פועל

## שידור התרעת DSC distress
1. [ ] **הרימו את המכסה הקפיצי מעל כפתור ה-DISTRESS האדום**
2. [ ] להתרעה מיועדת: בחרו את מהות המצוקה (אם יש זמן)
3. [ ] **לחצו והחזיקו את הכפתור (כ-5 שניות) עד שהוא משדר**
4. [ ] המכשיר מתכוונן אוטומטית ל-Ch 16 — **המשיכו בקריאת קול MAYDAY**
5. [ ] המתינו לאישור קליטה; ההתרעה חוזרת עד שמאושרת

## קריאת DSC פרטנית / שגרתית
- [ ] הזינו את ה-MMSI של התחנה האחרת (או בחרו מהספרייה)
- [ ] בחרו את ערוץ העבודה המוצע
- [ ] שדרו; כשמאושר, שני המכשירים עוברים לערוץ העבודה לשיחת קול

## ביטול DSC distress בשוגג
- [ ] **אל תכבו סתם** — בטלו כראוי
- [ ] אפסו את ההתרעה במכשיר, ואז שדרו ביטול קולי ב-Ch 16:
```
ALL STATIONS x3, THIS IS <boat name x3> <MMSI/callsign>
CANCEL MY DISTRESS ALERT OF <time> UTC. <boat name>. OUT
```

## הערות ומלכודות
- DSC distress שולח את **הזהות והמיקום (GPS)** שלכם דיגיטלית תוך שניות —
  זו הדרך המהירה והאמינה ביותר להרים אזעקה; השתמשו בה קודם, ואז בקול.
- אין כניסת GPS = אין מיקום בהתרעה; **ודאו שה-GPS מחובר**.
- התרעות בשוגג קורות — תמיד בטלו אותן; לעולם אל תשאירו אחת ללא טיפול.
