# צורות יום (Day Shapes) (COLREGs)

- **קטגוריה:** COLREGs — מדריך מהיר
- **סגנון:** עיון (Reference)
- **מתי להשתמש:** זיהוי כלי שיט ביום; הצגת הצורות הנכונות בעצמכם.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> עזר זיכרון בלבד — ראו חוקים 24–30 לפירוט מלא. הצורות שחורות.

| צורה | משמעות |
| --- | --- |
| **כדור** (אחד, קדמי) | **בעגינה** |
| **שלושה כדורים** (אנכי) | **על שרטון** |
| **חרוט, קודקוד למטה** | **כלי מפרש גם במנוע** (motor-sailing) |
| **שני חרוטים, קודקודים נפגשים** (כמו מעוין של שני ▲▼) | **כלי העוסק בדיג** |
| **כדור–מעוין–כדור** (אנכי) | **מוגבל ביכולת התמרון (RAM)** |
| **שני כדורים** (אנכי) | **שאינו בשליטה (NUC)** |
| **גליל** | **מוגבל בשוקע (Constrained by draft)** |
| **מעוין** (על עצם נגרר) | **גרירה מעל 200m** |
| **סל** | כלי קטן בדיג (מסורתי) |

## אלה שיאכטה באמת משתמשת בהן
- [ ] **כדור עגינה** — הניפו כדור שחור קדמי בכל עגינה ביום. **עשו זאת** —
  זה נשכח לעיתים קרובות וזה החוק.
- [ ] **חרוט מנוע (קודקוד למטה)** — הציגו כשמפליגים **וגם** מפעילים את המנוע.

## הערות ומלכודות
- כדור העגינה הוא צורת היום הנשכחת ביותר בכלי קטן — החזיקו אחד
  על הסירה והניפו אותו בכל עגינה.
- לצורות יום יש **אותה משמעות** כמו לאורות הלילה המקבילים.
