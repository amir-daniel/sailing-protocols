# Day Shapes (COLREGs)

- **Category:** COLREGs quick-reference
- **Style:** Reference
- **When to use:** Identifying vessels by day; showing the correct shapes yourself.

> Memory aid only — see Rules 24–30 for full detail. Shapes are black.

| Shape | Meaning |
| --- | --- |
| **Ball** (one, forward) | **At anchor** |
| **Three balls** (vertical) | **Aground** |
| **Cone, point down** | **Sailing vessel also under engine** (motor-sailing) |
| **Two cones, points together** (like a diamond of two ▲▼) | **Vessel engaged in fishing** |
| **Ball–Diamond–Ball** (vertical) | **Restricted in Ability to Manoeuvre (RAM)** |
| **Two balls** (vertical) | **Not Under Command (NUC)** |
| **Cylinder** | **Constrained by draft** |
| **Diamond** (on a towed object) | **Tow exceeding 200m** |
| **Basket** | Small vessel fishing (traditional) |

## The ones a yacht actually uses
- [ ] **Anchor ball** — hoist a black ball forward whenever anchored by day. **Do it** —
  it's frequently forgotten and it's the law.
- [ ] **Motoring cone (point down)** — show when sailing **and** running the engine.

## Notes & gotchas
- The anchor ball is the single most-forgotten day shape on small craft — keep one
  aboard and hoist it every time you anchor.
- Day shapes carry the **same meaning** as the equivalent night lights.
