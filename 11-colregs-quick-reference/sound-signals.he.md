# אותות קול (Sound Signals) (COLREGs)

- **קטגוריה:** COLREGs — מדריך מהיר
- **סגנון:** עיון (Reference)
- **מתי להשתמש:** בראיית כלי שיט אחר (תמרון) ובראות מוגבלת.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> עזר זיכרון בלבד — ראו חוקים 32–37. **·** = צפירה קצרה (~1 שנ'), **▬** = צפירה ארוכה (4–6 שנ').

## אותות תמרון ואזהרה (Rule 34) — כלים בראייה
| אות | משמעות |
| --- | --- |
| **·** (קצרה אחת) | "אני משנה כיוון **ימינה (to starboard)**" |
| **· ·** (שתיים קצרות) | "אני משנה כיוון **שמאלה (to port)**" |
| **· · ·** (שלוש קצרות) | "אני **מפעיל הנעה לאחור (astern)**" |
| **· · · · ·** (חמש או יותר קצרות/מהירות) | **סכנה / "אני בספק שאתה פועל מספיק"** |

### עקיפה בתעלה צרה (Rule 34)
| אות | משמעות |
| --- | --- |
| **▬ ▬ ·** | "בכוונתי לעקוף **בצדך הימני (starboard)**" |
| **▬ ▬ · ·** | "בכוונתי לעקוף **בצדך השמאלי (port)**" |
| **▬ · ▬ ·** | (הכלי הנעקף) "**מסכים (Agreed)**" |

### פנייה עיוורת
| אות | משמעות |
| --- | --- |
| **▬** (ארוכה אחת) | התקרבות ל**פנייה עיוורת / מכשול**; נענה בידי כל כלי מעבר |

## ראות מוגבלת (Rule 35) — משמיעים במרווחים (≤2 דק')
| כלי שיט | אות |
| --- | --- |
| **ממונע, מתקדם** | **▬** (ארוכה אחת, כל ≤2 דק') |
| **ממונע, בתנועה אך עצור** | **▬ ▬** (שתיים ארוכות) |
| **מפרש / דיג / RAM / NUC / מוגבל / גורר** | **▬ · ·** (ארוכה + שתיים קצרות) |
| **בעגינה** | צלצול מהיר של **פעמון** ~5 שנ' כל ≤1 דק' (ניתן להוסיף **· ▬ ·**) |
| **על שרטון** | פעמון כמו בעגינה, **בתוספת 3 הקשות פעמון נפרדות** לפני ואחרי |

## הערות ומלכודות
- **חמש צפירות קצרות = "איני מבין את כוונותיך / סכנה"** — השתמשו מוקדם
  כשכלי אחר אינו נותן זכות קדימה.
- כלי מפרש בערפל משמיע **ארוכה אחת + שתיים קצרות** (כמו כמה כלים מיוחדים).
