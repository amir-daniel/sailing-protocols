# אורות ניווט (COLREGs)

- **קטגוריה:** COLREGs — מדריך מהיר
- **סגנון:** עיון (Reference)
- **מתי להשתמש:** זיהוי כלי שיט בלילה / בראות מוגבלת, והצגת האורות
  הנכונים בעצמכם.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> עזר זיכרון בלבד — למדו את החוקים המלאים (20–31) ואת הקשתות.

## קשתות האורות (הבסיס)
- **Sidelights:** אדום (port) וירוק (starboard), כל אחד מציג **112.5°**
  מהחזית ועד מעט מאחורי הזרוע (beam).
- **Sternlight:** לבן, **135°** המכסה את הירכתיים.
- **Masthead/steaming light:** לבן, קשת קדמית **225°** (מעל קשת ה-sidelights).
- **All-round light:** לבן/אדום/ירוק, **360°**.
- Sidelights + sternlight יחד = **360°** סביב הכלי.

## מה תראו בדרך כלל
| כלי שיט | אורות |
| --- | --- |
| **ממונע בתנועה** (<50m) | Masthead (לבן, קדמי), sidelights, sternlight |
| **ממונע ≥50m** | **שני** אורות masthead (השני גבוה ואחורי), sidelights, sternlight |
| **כלי מפרש בתנועה** | Sidelights + sternlight (**ללא** masthead steaming light) |
| מפרש — אופציונלי | ניתן להוסיף all-round **red-over-green** ב-masthead |
| **מפרש במנוע ("motor-sailing")** | נחשב לממונע — הציגו masthead light + חרוט ביום |
| **כלי בעגינה** (<50m) | אור **all-round white** אחד (קדמי); ≥50m מציג שניים |
| **כלי על שרטון** | אור/ות עגינה **בתוספת שני all-round red** בקו אנכי |

## כלים מיוחדים — אורות all-round אנכיים
| כלי שיט | אורות (מלמעלה→למטה) |
| --- | --- |
| **שאינו בשליטה (NUC)** | **Red over Red** ("red over red, the captain is dead") |
| **מוגבל ביכולת התמרון (RAM)** | **Red / White / Red** |
| **מוגבל בשוקע (Constrained by Draft)** | **Three Red** בקו אנכי |
| **דיג (trawling)** | **Green over White** |
| **דיג (אחר)** | **Red over White** ("red over white, fishing at night") |
| **כלי נתב (Pilot)** | **White over Red** ("white over red, pilot ahead") |
| **גורר (tow >200m)** | גורר: 3 אורות masthead + אור גרירה צהוב מעל sternlight |

## חישוב הכיוון (aspect)
- **רואים אדום וירוק + לבן מעל** → הכלי בערך **חזית לחזית** (ממונע).
- **רואים ירוק + לבן** → אתם מביטים בצדו ה**ימני (starboard)**.
- **רואים אדום + לבן** → אתם מביטים בצדו ה**שמאלי (port)**.
- **רואים אור לבן בלבד** → כנראה **sternlight** (אתם עוקפים) או
  כלי עוגן/קטן — נהגו בזהירות.

## הערות ומלכודות
- כלי מפרש **אינו** מציג steaming light לבן קדמי; אם ראיתם אחד, התייחסו
  אליו כממונע (ייתכן שהוא במנוע).
- למדו את מנמוניקות הצבע — הן שליפה מהירה ב-0200 בנתיב שיט.
