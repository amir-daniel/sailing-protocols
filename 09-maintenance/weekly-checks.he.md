# בדיקות שבועיות

- **קטגוריה:** תחזוקה
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** שבועית בשיוט/מעבר ארוך, או לפני כל סוף שבוע של שימוש.
- **צוות:** סקיפר או מהנדס.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## מנוע ומכניקה
- [ ] בדקו/נקו את ה-**raw-water strainer**
- [ ] בדקו רצועות מקרוב; בדקו מתח
- [ ] בדקו דליפות דלק/שמן/נוזל קירור ומלאו לפי הצורך
- [ ] נקזו מים מה-fuel pre-filter/separator
- [ ] גירוז/שימון לפי לוח הזמנים של המנוע

## חשמל
- [ ] בדקו דקי מצבר; נקו והגנו
- [ ] מלאו מצברים (flooded); בדקו שמאובטחים ומאווררים
- [ ] בדקו את כל אורות הניווט והפנים

## כלונסאות ומפרשים
- [ ] בדיקת כלונסאות מפורטת: **swage terminals, קצות spreader, clevis/split pins**
- [ ] בדקו ש-reefing lines, blocks ו-cars נעים בחופשיות
- [ ] בדקו מפרשים לנזק תפר/UV; ציינו תיקונים נדרשים

## גוף ואינסטלציה
- [ ] **הפעילו ובדקו את כל ה-seacocks; בדקו hose clamps**
- [ ] בדקו משאבות bilge (ידנית + חשמלית) ו-float switch
- [ ] בדקו את מערכת המים המתוקים והמשאבה; חפשו דליפות

## בטיחות
- [ ] בדקו תוקף רקטות/EPIRB/PLB
- [ ] בדקו את ה-VHF (radio check) ואת ה-DSC
- [ ] בדקו דליפות במערכת הגז; בדקו תאריך הצינור

## הערות ומלכודות
- ה-raw-water strainer וה-fuel pre-filter הם שני הדברים שנסתמים בשקט
  ועוצרים את המנוע — שמרו אותם נקיים.
