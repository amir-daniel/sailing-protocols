# בדיקות יומיות (במעבר / בשימוש)

- **קטגוריה:** תחזוקה
- **סגנון:** בצע–אשר (DO–CONFIRM)
- **מתי להשתמש:** בכל יום שהסירה בשימוש, רצוי כל בוקר.
- **צוות:** סקיפר או מי שהואצל.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## מנוע ומערכות
- [ ] רמות שמן ונוזל קירור של המנוע
- [ ] **Bilge יבש; בדקו מים/שמן/דלק חדשים**
- [ ] מצב הרצועה; חפשו דליפות סביב המנוע
- [ ] מצב טעינת המצבר; הטעינה עובדת
- [ ] רמות מיכל דלק ומים; תכננו מילוי

## סיפון וכלונסאות
- [ ] סיירו על הסיפון: **כלונסאות, אביזרים, split pins, shackles מאובטחים**
- [ ] Halyards/sheets לשחיקה; שחררו/נתבו מחדש לפי הצורך
- [ ] מפרשים לנזק או שחיקה ב-spreaders/stanchions
- [ ] Fenders/warps מאוחסנים; שום דבר אינו נגרר

## בטיחות
- [ ] אפודי הצלה/רתמות/tethers במצב תקין
- [ ] Jackstays תקינים (אם מוקמו)
- [ ] אורות ניווט פועלים
- [ ] מטפים/עזרה ראשונה נגישים

## אדמיניסטרציה
- [ ] עדכון מזג אוויר התקבל
- [ ] היומן מעודכן; מיקום נקבע
- [ ] מים/מזון/דלק מספקים ליום

## הערות ומלכודות
- סיור כלונסאות יומי תופס כשלים קטנים (split pin שנפתח, halyard נשחק)
  לפני שהם הופכים לשבירת תורן או שברים.
