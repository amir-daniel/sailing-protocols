# עונתי — אחסון (Lay-Up) והחזרה לפעילות

- **קטגוריה:** תחזוקה
- **סגנון:** קרא–בצע (READ–DO)
- **מתי להשתמש:** סוף העונה (lay-up) ותחילת העונה (recommission).
- **צוות:** בעלים / מספנה.

> תרגום — ממתין לבדיקת אדם דובר עברית ומנוסה בשיט לפני הסתמכות מבצעית.

> פריטים קריטיים מודגשים ב**בולד**.

## Lay-up (סוף העונה)
### מנוע ומערכות
- [ ] החליפו שמן ומסנן מנוע; הריצו להסחרה לפני האחסון
- [ ] **הכינו את מערכת הקירור לחורף (antifreeze) אם קיפאון אפשרי**
- [ ] מלאו או טפלו במיכל הדלק כדי למנוע עיבוי/"חיידק דיזל"
- [ ] הסירו ואחסנו/טענו מצברים; שמרו טעונים לאורך החורף
- [ ] נקזו את מערכת המים המתוקים והוסיפו antifreeze לא רעיל

### גוף, כלונסאות ומפרשים
- [ ] העלו ליבשה; שטפו בלחץ ובדקו את הגוף, הכובד, ההגה
- [ ] בדקו anodes; ציינו צרכי antifouling
- [ ] הסירו מפרשים; שטפו, ייבשו, בדקו, אחסנו יבשים
- [ ] הורידו או בדקו את התורן/הכלונסאות לפי לוח הזמנים שלכם
- [ ] **אווררו את הפנים; מנעו לחות/עובש** (פתחו תאים, השתמשו במסיר-לחות)

### ציוד בטיחות
- [ ] ציינו תאריכי שירות/תפוגה: liferaft, רקטות, EPIRB, מטפים, אפודי הצלה

## Recommission (תחילת העונה)
- [ ] הפכו את הכנת החורף (נוזל קירור, מערכת מים, דלק)
- [ ] הרכיבו מצברים טעונים; בדקו את כל החשמל והאלקטרוניקה
- [ ] בצעו antifoul; החליפו anodes; בדקו ש-seacocks פועלים בחופשיות
- [ ] **בדיקת כלונסאות מלאה לפני הצבה/מתיחה; בדקו את כל ה-terminals**
- [ ] הרכיבו מפרשים; בדקו running rigging
- [ ] שרתו/החליפו ציוד בטיחות שפג תוקפו; **liferaft שורת**
- [ ] מנוע: בדקו רמות/רצועות/impeller; הריצו מבחן ואשרו מי קירור
- [ ] ניסוי ים לפני המעבר האמיתי הראשון

## הערות ומלכודות
- נזק קיפאון (heat exchanger/block סדוק) הוא הטעות היקרה הקלאסית של lay-up — הכינו את מערכת הקירור לחורף כראוי.
- הזמינו שירות liferaft ורקטות מוקדם; התורים מתמלאים לפני העונה.
